# JKA ES

## 企業研究メモ（経営企画編）

- 管理部門 → 経営企画部 → 企画課（志望）
- 選考スケジュール：3月からES受付 → 4月中旬グループ面接 → 個別面接 → 最終面接（対面） → 5月下旬内々定
- 配属：最初の配属はどこになるか不明（いろんな部署の可能性あり）。5月末に配属先発表

### 経営企画部 業務内容

1. **財団運営業務**：中期計画の策定・管理、年度方針および予算方針の策定・管理
2. **シンクタンク業務**：競輪・オートレース売上分析集計、経営分析、競輪・オートレースに関する調査（アンケート・インタビュー）
3. **企画調整業務**：他公営競技団体との連絡調整
4. **調整研究業務**：競輪とデジタルとの融合、競輪選手の競走ユニリニューアルデザイン制作、社内AIシステム導入による業務改善調査
5. **振興調整業務**

### シンクタンク機能

- 定量調査（アンケート）、定性調査（インタビュー）、現地調査
- 売上分析・仮説検討から新たな施策提案に至る

### JKAの特徴

- 株式会社ではないためKPIが利益追求ではない（売上を還元する組織）
- 経産省への説明責任（公平・公正）が重要
- 中期経営計画：策定に1年半かかった。月次売上報告が月初にある。

### インターンメモ

- 分析が良かった点、調べた部分も良かった点を評価
- 改善点：懸念点についての補足をしておけば良かった
- 客観的に自分の案・意見を批判的に見ることが重要
