→ **「契約者責任×独立判断×シニア集団×新卒直配置」の4点を同時に持つのはMYAMのみ**。

### 固有性 2: 【独自ROIC-WACCフレームワーク×ESG-DCF統合】

- MYAM は**ESG格付をDCF割引率に直接反映**(高評価-A%/低評価+B%)。他社はスコア化止まり。
- 約400銘柄独自ESG分析・対話シート作成 → Alpha源泉への統合構造が明確。
- 「理想は企業自身が安い理由を解消」(面談会証言) = **受動的スクリーニングではなく能動的企業変革モデル**。

### 固有性 3: 【営業と運用の完全分離（構造的担保）】

- 面談会証言: **「ファンドを売るからこの株を見といて、ということはない」**
- 販売ニーズ（防衛株等）は商品部経由で間接化、運用判断の独立性が担保
- AM-One は親会社依存、野村/大和は販売チャネル強いためこの純粋性は MYAM 固有

### 固有性 4: 【純流出状態＝「課題先進」企業】

- 唯一の純流出(▲22億円)だが、
- **外部顧客拡大KPIが経営戦略の中心** = 若手が貢献できる余地最大
- 他7社(全て純流入)は既存モデルの延長、MYAMは**変化の主体**

```
(2) 販売会社持たず地銀チャネル依存 → 新NISA時代の個人獲得で劣位
受け: その課題解決が自分の仕事。卒論(POSデータ)・ステーブルコイン分析の
「不完全データから実態を読む」思考は、販社ニーズの先読み・商品設計に直接活かせる
```
