```

```

## 記入ガイド

### 企業固有性に書くべきもの
- 運用哲学・投資戦略の独自性
- 商品ラインナップの特徴
- 企業文化・組織特性
- 市場ポジションや規模の特徴
- 最近の戦略変化

### 書いてはいけないもの
- 「顧客志向」「社員の成長を大切に」等の一般論
- 他社でも言えること
- 出典不明の主観的評価

### brief を更新するタイミング
- 新しい情報が入ったとき（IR・OB訪問後・説明会後）
- ES執筆前に内容が古い場合
- 面接後に「ここが薄かった」と気づいたとき
