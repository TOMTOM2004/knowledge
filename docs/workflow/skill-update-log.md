# スキル更新ログ

スキルへの変更履歴と、更新のトリガーとなった気づきを記録する。

---

## 記録フォーマット

```
## YYYY-MM-DD: [変更サマリー]

**対象スキル**: スキル名
**トリガー**: 面接フィードバック / レビュー見逃し / 通過パターン / その他
**気づき**: 何が問題だったか・何を改善したか
**変更内容**: スキルにどう反映したか
```

---

## ログ

（初期状態 — 運用開始後に記録を追加してください）

### 記入例

```
## 2026-03-28: es-rewrite-patterns に字数調整パターンを追加

**対象スキル**: es-rewrite-patterns
**トリガー**: 面接でフィードバック
**気づき**: 300字→200字への削減パターンが不足していた。
           前置きを削る方法が具体的でなかったため、
           「どこを削るか」判断に迷うことがあった。
**変更内容**: 「200字→300字の増量」「400字→300字の削減」パターンを追加。
           具体的な削除対象（接続詞・形容詞・定型締め）を明示した。
```
