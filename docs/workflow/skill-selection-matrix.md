# スキル選択マトリクス

「どの企業・職種・ステージ・目的」に「どのスキルを使うか」を一覧化。

---

## 使い方

1. 「何をしたいか」で行を選ぶ
2. 「企業の業種」「志望職種」「選考ステージ」で列を絞る
3. ◎=必須、○=推奨、△=任意、空=不要

## クイックリファレンス（目的別最短経路）

### 「ESを最初から書く（AM企業・運用職）」
```
es-writer → domains/finance-common + industries/asset-management-fit + roles/fund-manager-fit
         + question_types/motivation + gakuchika + audiences/junior-humanities
```

### 「ES提出前に最終チェック」
```
es-review-protocol → question-fit-reviewer + readability-reviewer + company-fit-reviewer
                   → editor-refiner（修正）
```

### 「1次面接を翌日に控えた準備」
```
interview-prep → stages/first-interview-mode
→ skeptical-interviewer → interview/motivation + gakuchika-answer-patterns
```

### 「最終面接（経営幹部）の前日準備」
```
stages/final-interview-depth-mode → interview/rebuttal-handling
→ interview/role-why-us-answer-patterns → skeptical-interviewer
```

### 「競合2社と比較して志望動機を強化」
```
company-comparison-framework → industries/<業種>-fit → research/asset-management-research-lens
```
