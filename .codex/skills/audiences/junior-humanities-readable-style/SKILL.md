# junior-humanities-readable-style

## 対象読者プロファイル
- 社会人1〜3年目の文系出身の人事担当者
- 金融・IT専門知識は持っていない
- 1つのESを3〜5分で読む
- 「ちゃんと答えているか」「具体性があるか」「人柄が伝わるか」を判断する

## 表現調整ルール

### 禁止表現と言い換え
| 元の表現 | 言い換え |
|---------|---------|
| KPI達成 | 設定した数値目標の達成 |
| PDCAを回す | 計画・実行・振り返りのサイクルを繰り返す |
| アジャイル | 小さく試しながら改善を繰り返す手法 |
| AUM | 運用している資産の総額 |
| フィデューシャリー・デューティー | お客様の最善利益を最優先する義務 |
| バリュエーション | 企業の価値を数値で評価すること |
| ポートフォリオ | 複数の投資対象を組み合わせた運用全体 |
| スキーム | 仕組み・枠組み |
| DX | デジタル技術を活用した業務・価値提供の変革 |
| 内製化 | 外部に委託していた作業を自社で行うこと |

### 文章スタイル調整
- 1文は50字以内を目標
- 結論を最初の1〜2文に持ってくる
- 「〜と思います」より「〜と考えています」「〜と確信しています」
- 箇条書きより流れのある文章（ESは特に）
- 接続詞の多用を避ける（「また」「さらに」「そして」の連続はNG）

### 確認チェックリスト
- [ ] 金融・IT専門用語が説明なしに使われていないか
- [ ] 1文が60字を超えていないか
- [ ] 冒頭3行で「この人が何をした人か」が伝わるか
- [ ] 主語が不明確な文がないか
- [ ] 「〜を通じて」が3回以上連続していないか
