# hr-readable-style

## 対象読者プロファイル
- 採用担当として毎年数百枚のESを読んでいる
- 業界知識はあるが深い専門性はない
- 「他の候補者と何が違うか」を常に比較しながら読む
- 「使い回しES」を見抜くプロ
- 人物像・一貫性・成長性を評価する

## 差別化のポイント

### 人事が「よくある」と感じるパターンと対処
| よくあるパターン | 差別化の方向 |
|--------------|-----------|
| 「チームワークを学んだ」 | 具体的なチーム内での役割と変化を示す |
| 「主体的に取り組んだ」 | 誰かに言われずに始めた具体的な行動を書く |
| 「貴社でグローバルに活躍したい」 | どの業務でどの地域かを具体化する |
| 「課題を解決した」 | 課題の発見方法・解決アプローチの独自性を示す |

### 人事が評価するポイント
1. **一貫性**: ガクチカ→自己PR→志望動機が同じ人物像で貫かれているか
2. **具体性**: 数字・名前・固有の場面が登場するか
3. **成長性**: この人は入社後どう成長しそうかのイメージが湧くか
4. **本気度**: この会社・職種を本当に調べた上で書いているか

### 文章スタイル調整
- 冒頭で他のESと違う「フック」（印象に残る具体事実）を置く
- ガクチカは活動名ではなく「自分の行動」を主語にする
- 志望動機は「企業名を変えれば別の会社にも送れる文」を除去する

### 確認チェックリスト
- [ ] 「企業名を変えれば他社でも使える文」がないか
- [ ] ガクチカが「活動の説明」で終わっていないか（自分の行動が主役か）
- [ ] 全設問を通じて同じ強み・価値観で一貫しているか
- [ ] 数字・固有名詞・具体的な場面描写があるか
