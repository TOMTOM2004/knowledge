# banking-fit

## 前提
finance-commonの観点を前提に、銀行業界固有の評価基準を追加する。

## 銀行業界で評価される特性

### 業界固有の評価軸
1. **長期リレーション構築力**
   - 顧客との長期的な信頼関係を構築・維持できるか
   - 「取引」ではなく「パートナーシップ」の視点
2. **課題発見・ソリューション提案力**
   - 顧客の言葉の裏にある本質的な財務課題を発見できるか
   - 「御用聞き」ではなく「課題解決者」であれるか
3. **泥臭さ・粘り強さ**
   - 法人営業は断られてから始まる仕事
   - 短期間での成果より継続的なフォローを評価
4. **規制・コンプライアンスへの感度**
   - 銀行法・マネロン規制・反社チェック等の重要性の理解

### 銀行内のコース別評価軸

| コース | 追加評価軸 |
|--------|-----------|
| 法人営業 | 企業財務理解・M&A/財務コンサル志向 |
| リテール | 個人の生涯設計への関心・きめ細かい対応 |
| マーケット | 市場分析・スピード判断・リスク管理 |
| 信託・資産管理 | 相続・遺言・不動産への専門性 |
| IT・デジタル | domains/digital-common 参照 |

### ES・面接チェックポイント

**志望動機で必ず確認すること**
- [ ] 「間接金融としての銀行の機能」を理解しているか（預金→融資の資金仲介）
- [ ] 証券・AM・保険との機能差が明確か
- [ ] 「なぜメガ・地銀・信託・信金か」の選択理由が具体的か
- [ ] 法人/リテールのどちらに重心を置くかが明確か

**NGパターン**
- 「銀行は社会インフラだから」（どのメガでも言える）
- 「グローバルに活躍したい」（国内法人営業の現実と乖離）
- 「融資で企業の成長を支える」（証券・AM・コンサルでも言える）

## 競合比較の視点
- メガ3行の差別化（MUFG/SMBC/みずほのそれぞれの強み）
- 信託銀行の独自機能（遺言信託・不動産管理・年金）
- 地銀との違い（地域密着vs広域対応）
