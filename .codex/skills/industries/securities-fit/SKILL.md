# securities-fit

## 前提
finance-commonの観点を前提に、証券業界固有の評価基準を追加する。

## 証券業界で評価される特性

### 業界固有の評価軸
1. **市場・情報への高感度**
   - 相場変動・経済ニュースをリアルタイムで追う習慣
   - 「情報を仕入れて価値に変換する」プロセスの理解
2. **スピード判断・実行力**
   - マーケット系は特に素早い意思決定が求められる
   - 「考えすぎる」より「仮説→実行→修正」のサイクル
3. **直接金融への使命感**
   - 企業の資金調達（IPO・公募増資・債券）を支える機能
   - 投資家と企業を繋ぐ「マーケットの橋渡し役」
4. **数字・パフォーマンスへの執着**
   - 結果が数字で見える環境を楽しめるか

### 証券内のコース別評価軸

| コース | 追加評価軸 |
|--------|-----------|
| リサーチアナリスト | 企業分析・モデリング・論理的文章 → roles/investment-analyst-fit 参照 |
| セールス（機関投資家向け） | 信頼構築・深い分析提供・長期フォロー |
| マーケット・トレーディング | 瞬時判断・リスク管理・数値センス |
| IB（投資銀行） | M&A・資本市場の知識・高密度作業耐性 |
| AM（証券系資産運用） | asset-management-fit と組み合わせる |

### ES・面接チェックポイント

**志望動機で必ず確認すること**
- [ ] 「直接金融vs間接金融」の機能差を理解しているか
- [ ] 銀行系AMとの違いを語れるか（証券独自のカルチャー）
- [ ] 「なぜこの証券会社か」の固有根拠があるか（野村/大和/SMBC日興等）
- [ ] 変動の激しい環境への適応意志が示されているか

## 競合比較の視点
- 日系大手3社の差別化（野村/大和/SMBC日興）
- 外資系証券との違い（報酬体系・仕事内容・文化）
- ネット証券との差別化（対面サービスの価値）
