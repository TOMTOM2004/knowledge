# asset-management-fit

## 前提
finance-commonの観点を前提に、AM業界固有の評価基準を追加する。

## AM業界で評価される特性

### 業界固有の評価軸
1. **受託者責任（フィデューシャリー・デューティー）への理解**
   - 顧客の最善利益を優先する姿勢
   - 利益相反への感度
2. **長期・複利思考**
   - 短期の変動より長期のリターン最大化を重視する視点
   - 「10年後・20年後を見据える」姿勢
3. **情報の非対称性への対処**
   - 一般投資家が知りえない情報・分析を提供することへの使命感
4. **市場・経済への知的好奇心**
   - 相場・経済ニュースへの継続的関心（義務ではなく習慣として）

### AM業界内のコース別評価軸

| コース | 追加評価軸 |
|--------|-----------|
| 運用（ファンドマネージャー） | 判断力・決断力・ポートフォリオ思考 → roles/fund-manager-fit 参照 |
| リサーチ（アナリスト） | 深掘り・数値モデル・企業分析 → roles/investment-analyst-fit 参照 |
| 営業（機関投資家・リテール） | 課題把握・提案力・信頼構築 |
| IT・デジタル | 金融×IT の掛け算理解 → domains/digital-common 参照 |

### ES・面接チェックポイント

**志望動機で必ず確認すること**
- [ ] 「運用業とは何か（信託報酬モデル・受益者利益）」を理解した上で書かれているか
- [ ] 銀行・証券との機能差が明確に語れているか
- [ ] 「長期×複利」の思想への共感が具体的に述べられているか
- [ ] AUM規模・競合との差別化が1点以上具体的に言及されているか

**NGパターン**
- 「投資に興味があるから」（個人投資家との混同）
- 「安定した仕事がしたい」（AM業界の特性との乖離）
- 「多くの人の資産を守りたい」（守るではなく「殖やす・運用する」が適切）

## 競合比較の視点（「なぜここか」の論拠）
- 独立系 vs 銀行系 vs 生保系 vs 外資系の違い
- 運用哲学（アクティブ vs パッシブ比率）の違い
- AUM規模・顧客層の違い（リテール重視 vs 機関投資家重視）
- 特定の強み領域（ESG・オルタナティブ・海外展開等）
