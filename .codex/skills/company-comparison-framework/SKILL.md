# company-comparison-framework

## 目的
複数の志望企業を同一基準で比較し、
- 各社の強み・弱みの整理
- 「なぜ競合ではなくこの会社か」の論拠作成
- 志望度整理・意思決定支援

に使う。

## 使う場面
- 「A社とB社を比較して志望動機に使いたい」と依頼されたとき
- 「なぜ競合ではないか」の説明を整理したいとき
- 複数内定の際の意思決定サポートに使うとき

## 比較フレームワーク

### 基本比較表
| 項目 | 企業A | 企業B | 企業C |
|-----|------|------|------|
| 事業領域 | | | |
| 売上規模 | | | |
| 強み | | | |
| 注力戦略 | | | |
| 求める人材像 | | | |
| キャリアパス | | | |
| 社風・文化 | | | |
| 志望度順位 | | | |

### 差別化ポイント分析
各社について:
1. 「他社にない特徴」を3点挙げる
2. 「自分のキャリアビジョンとの親和性」を5段階評価
3. 「なぜここでないといけないか」を1文で表現する

### ES・面接活用パターン
**「なぜ当社か」の回答構造**:
```
[競合との共通点認識] → [この会社固有の優位性/差別化] → [自分のビジョンとの接点]
```

例:
「銀行業界全体を見て、○○という点で御行に共通するものを感じました。その中でも特に御行は△△という点で他行と異なり、私が実現したい□□というキャリアに最も近い環境だと考えています。」

## 出力形式
- 比較表
- 差別化ポイント分析（企業ごと）
- ES・面接活用のための「この会社でなければならない理由」テンプレート
