# finance-common

## 目的
金融業界共通の評価観点・用語・思考様式を提供し、industry系・role系スキルの基礎層として機能する。

## 金融業界で評価される共通能力

### 思考特性
- **数字への親和性**: 感覚ではなくデータ・根拠で考える姿勢
- **リスク感覚**: リターンとリスクのバランスで判断する習慣
- **長期視点**: 短期の数字より長期の価値創造を重視する視点
- **誠実さ・正確性**: 顧客の資産・信頼を扱う職業倫理

### 共通語彙（ESで使用可能）
| 一般語 | 金融適合表現 |
|-------|-----------|
| お客様の役に立つ | 顧客の資産形成・財務目標の実現に貢献する |
| 情報収集が好き | 経済・市場情報のアンテナを張り、意思決定に活かす |
| 課題解決 | 顧客の財務課題に対するソリューション提供 |
| 分析が得意 | 定量データと定性情報を統合して判断する |

### 金融業界に共通するNG表現
- 「安定しているから」（志望動機として弱い）
- 「お金が好きだから」（職業倫理の観点で悪印象）
- 「金融に興味があります」（どの分野か不明）
- 「社会に貢献できる」（どの金融機能を通じてかが不明）

## 業界共通の面接深掘り論点
1. **なぜ金融か**: 製造業・コンサル・商社との違いを語れるか
2. **なぜこの機能か**: 運用/銀行/証券の機能差を理解しているか
3. **規制環境への理解**: フィデューシャリー・デューティー、金融庁規制
4. **デジタル化への認識**: フィンテック・AIによる金融変革への見解
