# banking-research-lens

## 目的
company-researcherスキルによる調査を銀行業界向けに深める追加観点を提供する。

## 銀行業界固有の追加調査観点

### 事業構造
- **コア業務の比重**: 融資業務 / 投資業務 / 手数料業務の比率
- **法人 vs リテールの比重**: どちらに注力しているか
- **海外展開の規模・拠点**: グローバルビジネスの実態
- **グループシナジー**: 証券・信託・保険との連携状況

### 中期経営計画の重点施策
- DX・システム刷新への投資額と方向性
- 収益改善策（低金利環境での収益多様化）
- 人材戦略（スペシャリスト化・外部採用）

### 競合比較（メガ3行 + 信託）
```
調査クエリ例:
[企業名] 法人融資 強み 中小企業 大企業 比率
[企業名] 海外展開 拠点 ASEAN アメリカ 規模
[企業名] DX 内製化 システム 投資 2024
三菱UFJ vs みずほ vs 三井住友 強み 比較 新卒
[企業名] 信託 信託業務 強み 残高
```

### コース別の追加調査
| コース | 追加で調べること |
|--------|--------------|
| 法人営業 | 主要取引先の業種・大企業vs中堅の比率 |
| リテール | 資産運用・投資信託販売への注力度 |
| マーケット | 市場部門の規模・運用残高 |
| IT・デジタル | ITシステムの内製化状況・DX子会社との関係 |
| 信託 | 年金・不動産・相続信託の残高 |

## 面接で使える差別化ポイントの発見方法
1. メガ3行の中での独自のポジション
2. 収益多様化戦略の具体性
3. 海外拠点の強さと戦略
4. デジタル化・DXへの本気度
