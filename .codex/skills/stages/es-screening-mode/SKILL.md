# es-screening-mode

## ES選考の特徴
- 読者: 人事担当者（採用経験3年以上）が多い
- 所要時間: 1枚3〜5分
- 比較対象: 同時期に数十〜数百枚
- 主な判断軸: 「説得力」「具体性」「一貫性」「志望度の本気感」

## 通過のための最低条件（チェックリスト）

### 必須条件（1つでも欠けると即落選リスク）
- [ ] 各設問に正面から答えている（設問ズレなし）
- [ ] 禁止表現なし（御社・様々な・思います等）
- [ ] 敬称が統一されている（貴社/貴行/貴グループ）
- [ ] 字数制限を守っている（制限の90%以上）
- [ ] 未記入・[TODO]がない

### 推奨条件（通過率を上げる）
- [ ] ガクチカに数字・定量表現がある
- [ ] 志望動機に企業固有の情報が含まれている
- [ ] 全設問を通じて人物像が一貫している
- [ ] 冒頭3行で「この人が何をした人か」が分かる

## 選考段階で重視される設問の優先順位
1. **志望動機**: 最も重視される（本気度の指標）
2. **ガクチカ**: 人物の具体的証拠
3. **自己PR**: 職種適合性の確認
4. **その他設問**: 加点要素

## この段階では深掘りしなくていいこと
- 完璧な論理構成より「読みやすさ」を優先
- 専門用語より分かりやすい言葉
- 完璧な数字より「具体的な場面描写」
