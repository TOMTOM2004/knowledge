# りそなグループ 競合プロファイル

## 1. グループ概要

- **りそなホールディングス** 傘下に りそな銀行（信託併営）・埼玉りそな銀行・関西みらいフィナンシャルグループ
- **証券子会社なし** — M&A助言・社債引受・IPO支援は外部証券会社に依存
- りそな銀行は **旧大和銀行（旧野村銀行）の流れを汲む、戦前から信託併営を続ける日本唯一の商業銀行**
- 店舗網は首都圏・関西圏中心で約400店。メガバンクとの規模差は大きい（みずほ銀行460店超＋信託独立店＋プラネットブース100超）
- グループアプリ 971万DL（グループ3行統合済み）
- りそな総合研究所（グループ会社）— 中小企業診断・コンサル中心。地域経済レポート（関西・埼玉）が主力

## 2. 法人営業（中堅中小向け）

### AI与信・デジタル融資
- デジタル施策は限定的。対面重視で大きな変化が少ない
- メガ（みずほ: UPSIDER買収、SMBC: Trunk）やふくおかFG（みんなの銀行）と比較すると出遅れ
- 2025年に「りそなグループアプリ for ビジネス」を開始（法人向けデジタル）

### 海外進出支援
- 海外拠点: 上海・香港・バンコク・ホーチミン・ハノイ・LA（**駐在員事務所中心**）
- 2025年にハノイ・LA開設。「リテールNo.1のグローバル化」を標榜
- 現地子会社銀行なし。提携銀行経由での情報提供が中心
- メガバンク（みずほ約40都市、MUFG約50カ国、SMBC約40カ国）との規模差は明確

### スタートアップ・VC
- りそなキャピタル＋**Fivotとベンチャーデットファンド共同設立**（2025年）
- デット中心（ベンチャーデット取扱開始2023年〜）
- ビジネスプラザ（大阪・東京・埼玉等）でスタートアップ交流拠点を運営
- IPO支援は限定的（証券子会社なし）

### 産業調査機能
- りそな総合研究所は中堅中小企業の診断・コンサルに特化
- 関西経済・埼玉経済レポートが主力。全産業横断のカバレッジはメガに劣る
- メガの産業調査部（特にみずほ産業調査部: 銀行本体内で唯一本格的）とは性質が異なる

## 3. 信託機能（★銀信一体の詳細）

### りそなの「信託併営」とは何か

りそな銀行は **日本で唯一の信託併営の商業銀行** であり、以下の信託機能が **銀行本体に内蔵** されている。メガバンクが別法人（信託銀行）で提供するサービスを、銀行RMが直接提案・実行できる。

| 信託サービス | 提供状況 | 実績・規模 |
|---|---|---|
| 自社株承継信託（議決権留保型） | **銀行RMが直接提案・実行** | オーナー企業の自社株承継で主力商品 |
| 自社株承継信託（遺言代用型） | **銀行RMが直接提案・実行** | 同上 |
| 不動産仲介・信託 | **銀行本体で実施（70年超の実績）** | 不動産仲介・資産管理・ファイナンスをワンストップ |
| 遺言信託 | **銀行窓口で直接受付** | 商品名「遺言信託」。信託金額1,000万円以上 |
| 企業年金 | **60年超の運用実績** | 制度設計・管理・資産運用を一体提供 |
| 証券代行（株主名簿管理） | **銀行本体で実施** | 上場企業の株主名簿管理、議決権集計 |
| 資産承継信託 | **「マイトラスト」「ハートトラスト」** | **50万円から対応可能**（低閾値で個人にも展開） |

### 銀信一体のメリット（りそな固有の強み）

1. **RM1人でのワンストップ提案**: りそなRMは融資→信託→不動産仲介を自分で完結できる。メガではBK RM→TB担当者への引き継ぎが必要
2. **スピード**: 別法人間の手続き（FW規制対応・同意取得・社内稟議）が不要。意思決定から実行まで早い
3. **顧客体験の一貫性**: 担当者が変わらないため、オーナーとの信頼関係が維持される
4. **低コスト層への信託提供**: PB最低1億円〜で中堅オーナーにも信託サービスを展開（みずほPBは推測10億円〜）。資産承継信託は50万円から
5. **FW規制の適用なし**: 銀行内部の信託部門のため、顧客情報共有に同意取得が不要。**即時の情報活用が可能**
6. **全支店で信託サービス提供可能**: 約400店の全支店で対応（メガは信託銀行の独立店舗のみ）

### 銀信一体の限界

1. **専門性の深度**: 銀行業務と信託業務を兼務するため、信託専業のプロ集団（みずほ信託銀行等）と比較すると専門性に限界がある可能性
2. **大型年金受託**: 中堅企業年金が中心。大型年金基金の受託は限定的（メガ系信託銀行はトップ3）
3. **証券代行の受託規模**: 受託企業数は信託専業3社に大きく劣後
4. **グローバル・カストディ**: 機能なし。グローバル資産管理はできない
5. **海外ネットワーク**: 海外拠点なし。海外に進出する企業の信託ニーズ（海外年金、クロスボーダーM&Aの株式実務等）には対応不可

## 4. 証券機能の欠如と外部依存

りそなグループには **証券子会社が存在しない**。以下の機能は外部証券会社への依存が必要:

| 機能 | 依存先（推測含む） | 影響 |
|---|---|---|
| M&Aアドバイザリー | 外部証券・M&A仲介会社 | 事業承継の先のM&A・IPOまで一気通貫で提供できない |
| 社債引受（DCM） | 外部証券 | 取引先企業の社債発行ニーズに自前で応えられない |
| IPO支援 | 外部証券 | IPO主幹事実績は限定的 |
| 株式引受（ECM） | 外部証券 | 同上 |
| M&Aに伴う株式実務（TOB代理等） | 外部対応 | 銀行融資→証券引受・M&A助言→信託の株式実務、という三位一体が組めない |

### 代替手段
- **承継ソリューション営業部（約200名・2023年新設）** が銀行内で事業承継コンサルを完結
- りそなキャピタル（事業承継ファンド）を運営
- ベンチャーデットはFivotと共同ファンドで取扱開始（2023年〜）

→ 事業承継の「入口」は強いが、「出口」（M&A実行・IPO・社債発行）は外部依存

## 5. 銀証信託連携の実装度

### りそなの構造的位置づけ

| 比較軸 | りそな（1法人内信託） | メガバンク（別法人型） |
|---|---|---|
| 法人格 | 銀行本体に信託部門を内包 | 信託銀行が別法人（FG直下） |
| 情報共有 | **FW規制適用なし → 即時** | FW規制対応必要（同意取得） |
| 顧客担当 | RM1人で完結可能 | BK RM → TB担当者へ引き継ぎ |
| 意思決定速度 | 1法人内で完結 → 高速 | 別法人間の稟議・引き継ぎに時間 |
| 専門性 | 銀行業務の付随機能として提供 | 信託専業のプロが専任対応 |
| グループ機能の幅 | 銀行＋信託（証券なし） | 銀行＋信託＋証券（三位一体） |

### メガバンクとの実装度差

- **みずほ**: 統一人事制度「かなで」（5社共通の職務等級・報酬・システム一元化）とプラネットブース（100拠点超で銀行内に証券社員常駐）で、別法人のデメリットを制度とインフラで克服。さらにオプトアウト方式でFW規制対応も効率化済み
- **MUFG**: 証券がモルガン・スタンレーとの合弁（60:40）のため意思決定に外資の承認が必要。2024年にFW違反で行政処分
- **SMFG**: グループ内に信託銀行がない（三井住友信託は完全別グループ・別上場）。構造的に相続・年金・不動産信託をグループ内完結できない

→ りそなの「1法人の便利さ」は真に強力だが、メガ（特にみずほ）は制度改革で連携速度の差を縮めつつある

## 8. 対メガバンク一般の比較ポイント（りそな面接で使える）

### 「なぜメガではなくりそなか」の論点

1. **銀信一体はりそなだけ**: メガバンクは全て信託を別法人で運営。FW規制対応・担当者引き継ぎ・稟議の時間コストが構造的に発生する。りそなはこれがゼロ
2. **中堅中小オーナーへのリーチ**: メガPBの閾値は高い（推測10億円〜）が、りそなは1億円〜で信託フルラインを提供。「資産家だけでなく、地域の中堅オーナーにも信託を届ける」という思想
3. **個人向け信託の充実**: 50万円からの資産承継信託はメガにない低閾値。リテール重視の姿勢
4. **不動産×融資のワンストップ**: 銀行本体で不動産仲介が可能（70年超の実績）。メガは別法人の信託銀行経由
5. **事業承継の専門部隊**: 承継ソリューション営業部200名は銀行内完結。メガはBKコンサル部＋TBコンサル部の連携が必要

### 「りそなの弱みをどう認識しているか」を聞かれた場合

- 証券機能がないため大型M&A・IPO・社債は外部依存 → だからこそ**承継の入口（信託・コンサル）に集中特化**できている
- 海外ネットワークは薄い → だからこそ**国内中堅中小への密着度**でメガに勝る
- 規模ではメガに及ばない → だからこそ**1法人の機動性とスピード**が活きる

### メガとの規模比較（参考）

- みずほFG純利益 1兆円超 vs りそなHD（規模は大幅に小さい）
- みずほ銀行 460店超 vs りそな約400店（ただし首都圏・関西集中）
- 海外: みずほ約40都市 vs りそな6拠点（駐在員事務所）
