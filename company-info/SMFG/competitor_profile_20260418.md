# SMFG 競合プロファイル

## 1. グループ概要

### 法人格構造
- **主要法人**: 3社＋小規模信託（SMBC / SMBC日興証券 / SMBC信託銀行）
- **SMBC信託銀行**: 旧シティバンク個人部門を承継した小規模行。三井住友信託銀行とは**完全別グループ・別上場**であり、SMFGグループ内に本格的な信託銀行は存在しない
- **持株会社の統制度**: 高い。トップダウン型の意思決定
- **BK/SC社長の関係**: 銀行主導。SMBC日興証券は100%子会社だが、2023年の相場操縦事件後に体制刷新

### デジタル先進性
- **Olive**: 預金・カード・証券を統合した個人向けプラットフォーム。SBI証券連携で2026年春にOlive Infinite開始
- **Trunk**: 2025年5月開始の中小企業向けデジタル総合金融プラットフォーム。口座開設最短翌営業日、振込手数料145円（業界最安水準）、30万社獲得目標（3年）
- **IT投資**: 8,000億円（うち500億円がAI）
- **生成AI**: IT投資の一環として積極活用推進中

### 関西基盤
- 三井住友銀行は旧住友銀行（大阪）の流れを汲み、関西圏に強固な顧客基盤を持つ
- 京都グローバル特化部署を2025年に新設（海外進出支援の関西拠点）

## 2. 法人営業（中堅中小向け）

### AI与信・デジタル融資
- **Trunk**（2025年5月〜）: 中小企業向けデジタル総合金融プラットフォーム
  - 口座・カードの統合プラットフォーム
  - 口座開設最短翌営業日
  - 振込手数料145円（業界最安水準）
  - Trunk経由で30万社獲得目標（3年）
- AI審査の詳細スペックは未公表（UPSIDER型のAI与信モデル内製化には至っていない）

### 事業承継・M&A仲介
- SMBC日興証券 ＋ 日本M&Aセンターとの連携
- SMBC系ファンドによる事業承継ファンドあり
- 不動産M&Aは限定的（グループ内に本格的信託銀行がないため）
- 三井住友信託銀行はSMFG外のため、信託連携は**構造的に不可**

### 海外進出支援
- 約40カ国に拠点展開
- ベトナム・インドなどアジア新興国に強み
- 2025年に京都グローバル特化部署を新設
- 「デジタル×グローバルの融合」を標榜

### 産業調査・知見提供
- **日本総合研究所**（グループ会社）がシンクタンク機能を担う
- SMBCマンスリー・レビュー / 産業調査レポートを法人向けに公開
- 政策提言に強い
- ただし銀行本体内の産業調査部はなく、RMとの直接協働度はみずほに劣る

### スタートアップ・PE・イノベーション
- **SMBC Edge**（旧SMBCベンチャーキャピタル）: 1,500億円ファンド。エクイティ＋事業開発支援
- シンガポール拠点を2026年春に開設予定
- BizDev支援付きVCとして、投資先の事業開発まで伴走する形態
- エクイティ投資中心（デット型のベンチャーファイナンスは限定的）

### ファミリーオフィス・PB・オーナー個人資産
- SMBC PBグループが富裕層向けサービスを提供
- PB最低預入額は非公表
- 銀行＋証券のクロスセルが基本モデル
- 三井住友信託銀行がSMFG外のため、遺言信託・不動産信託などをグループ内で提供できない（構造的弱点）

## 3. 証券機能（SMBC日興証券）

### DCM（債券引受）
- 2024年4-12月 公募社債引受額: **2.64兆円（3位圏）**
- SMFG系の金融債に強い（推測）
- みずほ（3.25兆円・首位）・大和（2.89兆円）に次ぐポジション

### ECM（株式引受・IPO）
- **2024年IPO主幹事: 23社（業界1位）**
- 中小型〜大型まで最多件数を獲得
- **バンク&トレード連携**: SMBCの法人基盤からIPO候補を発掘（ispaceやQPS研究所等の実績）
- 銀行連携でAI企業等の新興企業を積極的に発掘

### M&Aアドバイザリー
- 2024年Bloomberg金額ランキングで上位10位圏
- 国内中心で、クロスボーダー比率は低め
- SMBC連携で中堅企業に一定の実績
- バンカー体制は中規模

### ストラクチャードファイナンス
- SMBC系でプロジェクトファイナンス実績あり
- REIT・証券化商品は中位
- 航空機・船舶ファイナンスは限定的

### リテール・ウェルスマネジメント
- 全国約120拠点を展開（SMBC日興証券単体）
- 預り資産: 3位圏（推測60-70兆円）
- ファンドラップ残高: 2.87兆円+
- バンク&トレードでSMBCとのオンライン口座連携
- IFA対応あり
- 2025年からSBI証券との連携も開始（Olive経由）
  - ただしSBI証券（外部）との役割分担がSMBC日興との間で曖昧化するリスクあり

### グローバルマーケッツ
- 東京中心にシンガポール等に拠点
- 国内中心のセールス&トレーディング
- リサーチ体制は中規模

### 若手キャリアパス
- 初任給: 26万円前後（2025年入社）
- SMFG内での異動（BK↔SC）は可能だが限定的
- コース別採用（みずほのグループオープン型とは異なる）
- 海外駐在は限定的

## 4. 銀証信託連携の実装度（★信託なしの構造的課題）

### 連携構造の現状
- **主要法人**: SMBC + SMBC日興証券の**2社体制**（本格的信託銀行なし）
- SMBC信託銀行は旧シティバンク個人部門の小規模行であり、事業承継・年金・不動産信託等の本格的な信託機能は提供できない
- **三井住友信託銀行は完全に別グループ・別上場**（SMFG外）

### 信託不在がもたらす構造的弱点

| 領域 | 影響 |
|---|---|
| **事業承継** | 自社株承継信託（議決権留保型・遺言代用型）をグループ内で提供できない。外部信託銀行に依存 |
| **不動産信託** | 不動産仲介・信託をグループ内で完結できない |
| **遺言信託** | グループ内での遺言信託受託不可 |
| **企業年金** | 年金制度設計・管理・資産運用のグループ内一体提供不可 |
| **証券代行** | 株主名簿管理等の信託業務をグループ内で提供できない |
| **M&A後の株式実務** | TOB代理等、M&A完了後の株式実務処理をグループ内で完結できない |
| **PB・ファミリーオフィス** | オーナーの個人資産管理（遺言・不動産・承継）にグループ内信託を組み込めない |

### 信託窓口
- **グループ内に信託窓口なし**（SMBC信託銀行は小規模で機能限定的）
- SMBC日興の支店は約120拠点だが、銀行内同居（みずほのプラネットブース型）は限定的

### 顧客情報共有・CRM
- Olive基盤で**個人は統合進行中**（デジタル先行）
- **法人の銀証連携は発展途上**
- ファイアウォール規制に関する大きな違反報道はなし

### 銀行⇔証券の送客フロー
- バンク&トレード（オンライン口座連携）が主軸
- 銀行窓口での証券仲介
- 2025年からSBI証券も連携（Olive経由）
- ただしSMBC日興の約120支店と銀行の物理的同居は限定的

### 人事制度
- 各社独自の人事制度（みずほ「かなで」のようなグループ統一人事制度はなし）
- 出向ベースでのグループ横断異動
- コース別採用

### デジタル統合
- **Oliveが個人向け統合で最も先行**: 預金・カード・証券（SBI証券）の一元管理
- **Trunk**が法人向けデジタル金融で先行
- IT投資8,000億円（うち500億円がAI）
- デジタルUI/UXではメガバンク中最も進んでいるが、**法人の銀証信託連携のデジタル化とは別次元**

## 5. 強み・弱みサマリー

### 強み
1. **IPO主幹事 業界首位（2024年23社）**: バンク&トレード連携による新興企業発掘力
2. **デジタル先進性（Olive / Trunk）**: 個人向け金融プラットフォーム統合でメガバンク最先行。法人向けTrunkも先行
3. **IT投資規模（8,000億円）**: AI投資500億円を含む大規模投資で業務効率化・サービス高度化を推進
4. **SMBC Edge（1,500億円ファンド）**: BizDev支援付きVCとして事業開発まで伴走。シンガポール拠点も開設予定
5. **トップダウン型の迅速な意思決定**: 持株会社の統制度が高く、グループ戦略の実行速度が速い
6. **関西基盤**: 旧住友銀行の歴史的顧客基盤。京都グローバル特化部署の新設

### 弱み
1. **★グループ内に信託銀行がない（構造的・最大の弱点）**: 三井住友信託は完全別グループ。事業承継・年金・不動産信託・証券代行をグループ内で完結できない
2. **DCMではみずほ・大和に劣後**: 公募社債引受額2.64兆円は3位圏。首位みずほ（3.25兆円）に差
3. **クロスボーダーM&Aが弱い**: M&Aは国内中心。MUFGモルスタやみずほGreenhill のようなグローバルIB機能がない
4. **グループ統一人事制度なし**: みずほ「かなで」のような5社横断の統一人事制度・システムがない。横断キャリアの柔軟性で劣る
5. **銀行内証券（物理統合）が限定的**: みずほのプラネットブース（100拠点超・証券社員常駐）のような物理的ワンストップ体制がない
6. **SMBC日興証券の信頼回復途上**: 2023年相場操縦事件後の体制刷新中
7. **SBI証券との役割分担リスク**: Olive経由でSBI証券を接続する一方、SMBC日興との役割分担が曖昧化する懸念

## 6. 対みずほ比較ポイント

### SMFGが みずほに勝っている領域

| 領域 | SMFGの優位性 | 具体ファクト |
|---|---|---|
| **IPO主幹事件数** | 業界首位 | 23社（2024年）。みずほ19社（2位） |
| **デジタル統合（個人）** | Oliveでメガ最先行 | 預金・カード・証券の一元管理。みずほは外部連携（楽天・マネーフォワード）依存 |
| **法人デジタル** | Trunkで先行 | 2025年5月開始。みずほはUPSIDER連携だがプラットフォーム統合度では劣る |
| **VC規模** | SMBC Edge 1,500億円 | エクイティ＋BizDev支援。みずほBLUE DREAM Fund（243億円）より規模大 |
| **IT投資額** | 8,000億円 | うちAI 500億円。メガバンク中最大級 |

### みずほが SMFGに勝っている領域

| 領域 | みずほの優位性 | 具体ファクト |
|---|---|---|
| **信託機能（三位一体）** | SMFGには構造的に不可 | みずほ信託銀行がFG直下。SMFG内に本格的信託銀行なし |
| **DCM（債券引受）** | 12年以上連続首位 | 3.25兆円（2024年4-12月）。SMBC日興 2.64兆円 |
| **グループ統一人事「かなで」** | SMFGには類例なし | 5社の職務等級・報酬・人事システムまで一元化 |
| **プラネットブース** | 物理的銀証統合で圧倒 | 100拠点超に証券社員常駐。SMBC日興は銀行内同居が限定的 |
| **産業調査部** | 銀行本体内の調査機能 | RMと協働で取引先に入る。日本総研はグループ会社で距離あり |
| **クロスボーダーM&A** | Greenhill買収で内製化 | 15拠点のグローバルネットワーク。SMBC日興は国内中心 |
| **AI与信（ベンチャーデット）** | UPSIDER買収で内製化 | AI与信モデルをグループ内に保有。SMFGはエクイティ中心 |
| **FW規制コンプライアンス** | 大規模違反なし | SMBC日興は2023年相場操縦事件で信頼回復途上 |

### 面接で「なぜみずほであってSMFGではないか」に使えるロジック

> 「SMFGはOlive/TrunkのデジタルやIPO件数で先行していますが、**グループ内に信託銀行がないという構造的弱点**があります。事業承継・年金・不動産信託・M&A後の株式実務をグループ内で完結できません。みずほは**銀行・証券・信託の三法人を完全自前で持ち、統一人事制度「かなで」で5社の人事システムまで一元化**しています。デジタルは追いつける差ですが、信託の有無は組織構造の問題で、簡単には埋められません」
