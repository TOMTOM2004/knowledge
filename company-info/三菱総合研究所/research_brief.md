## 軸1: 事業・ビジネスモデル

三菱総合研究所（MRI）は1970年、三菱創業100周年記念事業として設立されたシンクタンク。東証プライム上場（3636）。連結売上1,215億円（FY2025/9）。五大シンクタンクの3位。

**2大事業セグメント：**

| セグメント | 売上 | 比率 | 特徴 |
|-----------|------|------|------|
| コンサルティングサービス（TTC） | 471億円 | 38.7% | 官公庁・民間向け調査研究・コンサルティング |
| ITサービス（ITS） | 744億円 | 61.3% | 三菱総研DCS中核。システム開発・BPO・データセンター |

顧客構成（TTC）：官公庁31% / 金融46% / 一般産業23%

注力成長領域：DX・GX（グリーントランスフォーメーション）・HX（人材・ヘルスケア）の3領域。「構想から実装までを一貫して担う」ビジネスモデルへの転換を推進中。

## 軸1-B: 収益構造の中身

- **TTCセグメント：** 官公庁・地方自治体からの政策立案・調査受託（プロジェクト型・フロー収益）が中心。大型政府案件は縮小傾向だが、民間DX戦略コンサルで補完しようとしている
- **ITSセグメント（三菱総研DCS）：** 人事給与BPO（PROSRV：受託約2,000社）・入試管理（miraicompass：全国1,910校）などストック型収益が安定
- **ストック型への転換方針：** 中計2026では「人的リソースに依存しないサービス型事業の拡大」が明言
- **収益特性のリスク：** TTCは大型案件完了後に受注残が急落する構造的リスクあり

## 軸4: 統計データ・社会データ分析コース vs 他コースの差別化軸

| コース | 特徴 | 自分との接点 |
|------|------|------------|
| **統計データ・データアナリティクス** | 官公庁データ・公的統計・AI/ML活用。政策の「数字」を作る側 | 卒論（POSデータ・傾向スコア）の手法が直結 |
| **社会データ分析** | 大規模社会調査・市場予測・生活者行動分析。社会現象の「測定」に特化 | 計量経済学・因果推論の視点が活かせる |
| リサーチコース | 政策立案・提言。調査・研究が中心 | 専門知識は活かせるがデータ分析特化ではない |
| DXコンサルティング | 企業のデジタル変革。ビジネスプロセス改善が中心 | データ分析より業務変革に軸足 |

**「なぜこのコースか」で使える観点：**
- 公的統計の精度向上は「見えないインフラ整備」であり、改善された統計が正しい政策立案に繋がって初めて社会に価値が生まれる
- 「分析して提言するだけでなく、使われる仕組みを設計する」という仕事哲学と合致
- 卒論テーマ（手取り増政策×消費行動の因果推定）がまさに政策効果の定量評価であり、業務と直結

## 軸6: 企業の強み・弱み（絶対評価）

### 強み

1. **政策立案力（圧倒的）**：官公庁との信頼関係は他社には真似できない。省庁の科学技術政策に20年以上関与。公的統計の改善委託を独占的に受注しているケースも多い。
2. **三位一体の統合力（唯一）**：リサーチ×コンサル×ITを同一ファームで実施できる。他社では「研究か実装か」に分かれがちだが、MRIだけが政策提言から社会実装まで一気通貫。
3. **社会データへの直接アクセス**：政府委託案件を通じて公的データ（医療・経済統計・国民生活調査等）に触れられる。統計・社会データ専門家を目指す上での独自環境。
4. **多専門領域の専門家集団**：医療・工学・環境・経済など理系背景のスペシャリストが競合より多い。文系×データサイエンスのブリッジ人材は差別化しやすい。
5. **DCS経由のストック型収益基盤**：人事給与BPO・入試管理など継続型事業で景気耐性が高い。

### 弱み・課題（貢献可能性）

1. **給与と激務のバランス**：平均残業43.9時間。「売上ノルマが高い割に給与が見合わない」という口コミが存在 → 業務効率化・AI活用推進で貢献できる余地
2. **専門性の深掘り難度**：1人5プロジェクト体制のため特定分野に集中しにくい → データ分析の型化・モジュール化で効率を上げる役割
3. **年功序列の残存**：マネージャーまでの約10年は年次進行が基本
4. **官公庁大型案件の構造的縮小**：民間DX案件のスケールアップが急務 → 入社後に民間DX拡大PJに参画できる

## 軸9: 面接対策・詰められ論点・逆質問候補

### 9-1. 詰められ論点と想定Q&A

**Q1: なぜシンクタンク・コンサルか（vs 金融・メーカー）**
「卒論で政策の因果効果を推定する中で、正しい統計が正しい政策を生むという構造に気づいた。企業で分析するより、社会全体の意思決定に使われる数字の精度を上げる仕事の方が社会へのインパクトが大きいと判断した」

**Q2: なぜNRI・JRIではなくMRIか**
「NRIはIT・民間コンサルが中心で公的統計の改善には関与しにくい。JRI・MURCは金融グループの色が強く中立的な政策提言に制約がある。大和総研は政府案件が少ない。MRIだけが官公庁の統計業務に深く関与しながら、実装まで一貫して担える三位一体の構造を持つ。自分の卒論テーマ（消費行動の政策効果推定）がそのまま業務に直結する企業はMRIだった」

**Q3: なぜ戦略コンサル（McKinsey等）ではないか**
「戦略コンサルは民間企業の利益最大化が中心。一方MRIは政策設計という公共的な意思決定支援が業務の核にある。社会全体の構造を変える政策の立案過程に関わりたいという志向と合致している」

**Q4: 統計・データ分析の経験は学部生としては浅くないか**
「卒論では傾向スコア・DID・IPWをPythonで実装し、理論と実装を往復させた。統計検定2級・GCI松尾研・GAの資格も取得済み。『深い理論家』ではなく『実装まで完結させる分析者』として貢献できる」

**Q5: マルチアサイン（複数PJ並行）に対応できるか**
「インターンで複数案件を並行処理した経験がある。タスクをリスト管理して可視化し、優先度と締め切りを常時把握する進め方を実践している。ただ、抱え込む傾向があるため、先読み報告と早期相談を意識的に習慣化している」（弱みの開示 + 改善中の姿勢）

**Q6: 専修大学という学歴でMRIは難しくないか**
「学歴より専門性の質で勝負している。計量経済学・因果推論の研究、データサイエンティストとしての長期インターン実務、生成AI活用・クラウドデータ基盤整備の経験は、院卒理系と比較しても差別化できる。MRIが求める文理融合のブリッジ人材として貢献できると確信している」

### 9-0. 選考フロー（確認済み）

**コンサルタント職の選考フロー（2026卒実績）：**

```
ES提出 → GD（グループディスカッション） → 1次面接 → 最終面接 → 内定
```

**GD詳細（要注意）：**
- 形式：学生6人 + 試験官2人、所要時間60〜90分
- 特徴：**役割分担・ホワイトボード使用禁止**（他社GDと異なる点）
- 冒頭：自己紹介 + 最近のニュース発表（1人ずつ）→ テーマ発表 → GD開始
- 成果物：発表用資料をGD時間内に作成し、最後に発表
- テーマ例：「大手通信事業会社の新規事業立案」「AIによる店舗販売への影響」「電気自動車普及後の社会変化」

**GD評価軸：**
| 評価軸 | 内容 |
|--------|------|
| 論理性 | 論理的思考。前提確認・タイムスケジュール設計ができるか |
| 現実性 | リアリティある提案。データ・エビデンスベースの思考 |
| 積極性 | 主体的発言。議論を活性化する動き |
| 協議性 | 他者意見の傾聴と統合。役割意識（禁止ではなく「意識」） |

**GD対策のポイント：**
- 役割宣言禁止のため、自然な形でファシリテーション・タイムキープ・書記を流動的に担う
- MRIらしい「エビデンスベース」思考を議論中に示す（感覚論ではなくデータや事例を引用）
- 冒頭のニュース発表：社会課題・統計・政策系のニュースを1つ準備（統計コース志望らしさを演出）

**最新動向（2025-2026）：**
- 三菱総研DCS × カオナビ：販売パートナー契約締結（人事給与BPO + タレントマネジメントの一体提供）
- 九電みらいエナジーへの蓄電池最適運用計画策定支援（GX領域の実装案件）
- AIエージェント活用型新規事業探索サービスを2026年1月に商用化（DX最重要施策の実装フェーズ）
