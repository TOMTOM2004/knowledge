## 軸5: キャリアパス

### 入社後の一般的ルート（インタビュー事例より）
- 入社時研修（クレジットカード事業基礎）→ 現場部門配属
- 2〜3年: 債権管理・オペレーション等で現場業務を学ぶ
- 3〜5年目以降: チャレンジキャリア制度（公募異動）や年1回の異動希望申告で経営企画・商品企画へ
- 実例: 伊藤氏（債権管理2年→経営企画）、尾﨑氏（法人カード3年→債権管理2年→経営企画）

### 中長期
- 経営企画・商品企画での実績積み上げ後、JR東日本グループの他社への出向・グループ横断プロジェクトへの参画も想定される
- 専門職・管理職の分岐は入社後5〜7年が目安 【要確認】

### 制度
- ジョブローテーション、チャレンジキャリア制度（公募型）
- 研修: クリティカル・シンキング、ビジネス・アナリティクス、マーケティング戦略基礎
- 資格取得支援（80以上の資格で受験料補助）、グロービス費用補助

## 軸6: 企業の強み・弱み

### 強み（ESで言及できる知識として）

| 強み | 内容 |
|------|------|
| Suicaオートチャージ独占 | 首都圏の交通インフラと決済を一体化できる唯一の存在。競合他社は構造的に代替不可 |
| JR東日本グループシナジー | 駅ビル（ルミネ・アトレ）優待、新幹線eチケット高還元（最大10%）、「JRE BANK乗車券4割引」等、交通特典は他社クレカと圧倒的差別化 |
| 会員基盤の安定性 | 592万人の首都圏ユーザー。定期券・通勤需要に紐づくため解約率が低い |
| データ資産 | 592万会員の行動・購買データ保有。CRM・クラスター分析・途上与信AI等に活用中 |
| JRE BANKによる拡張 | 2024年5月開始の銀行代理業で金融サービスの幅を拡大。JR東日本の信用力を背景に差別化可能 |

### 弱み・課題（貢献可能性の観点として）

| 課題 | 内容 | 貢献余地 |
|------|------|---------|
| 意思決定の遅さ | JR東日本という巨大親会社の存在によりスピード感が出にくい（口コミ複数） | 経営企画内で根拠ある提案を迅速に形式化し、意思決定を加速する役割 |
| 収益率の低さ | 営業利益率3.7%、ROA 0.2%と低水準。カード事業の構造的な利益率課題 | 新商品・手数料収益最適化の企画立案で収益改善に貢献 |
| DX推進の社内合意形成 | CRM施策推進時に「ターゲティングでリーチが減る」という営業部門からの抵抗が発生。データ活用の文化醸成が課題 | データで根拠を示し、社内ステークホルダーを動かす経験・スキルが活かせる |
| グループデータ統合未達 | ビューカード社内データのみ活用。JREグループ全体（Suicaデータ等）との統合が未達 | データ基盤設計・分析設計の上流から関与できる余地 |
| 競合激化 | 楽天カード・dカード・QRコード決済の台頭。フィンテックの参入リスク | Suicaの独自性を活かした差別化商品・施策の企画で対抗 |
| 属人的オペレーションの残存 | DX以前は途上与信も手動。まだ属人的なノウハウ依存が残る部門がある可能性 | 仕組み化・型化のアプローチで改善に貢献 |

## 軸7: エピソード適合マッピング

経営企画・商品企画開発職が重視する資質（折衝力・幅広い知識・課題解決・チェンジリーダー的姿勢）を基準に評価。

| エピソード | 評価 | コメント |
|-----------|------|---------|
| 03_インターン_財務資料自動化 | ○ | 「課題を構造化し仕組みで解決する」力は経営企画の業務改善・DX推進文脈で通じる。ただし「商品企画・新サービス創出」との直結度は低く、技術寄りの印象になりすぎないよう注意が必要 |
| 05_バイト_塾バイトリーダー | ○ | 「組織全体の品質を標準化・改善した」点が経営企画の横断的推進力と一致。脱属人化・ビジョン共有は商品企画でのプロジェクト推進にも応用可 |
| 08_学業_計量経済学ゼミ | ○ | データで根拠を示し意思決定を支援する姿勢は、経営企画でのKPI管理・事業計画の数値根拠づくりに通じる。ビューカードのCRM・DX課題への貢献文脈でも有効 |
| 04_インターン_分析設計失敗 | △ | 失敗からの改善・構造化は弱みの設問で有効。ただし「設計を先に固め、関係者に早期共有する」学びが経営企画の折衝スタイルと接続しているかを補強する必要がある |
| 16_自己分析_設計思想・AIオーケストレーション | △ | ビューカードのDX・属人的オペレーション改善の文脈では貢献余地として言及できる。ただし経営企画・商品企画はエンジニア職ではないため、「技術を使いこなす設計者」という位置づけを明確にしないと職種ミスマッチに見える |
| 06_課外活動_中学文化祭チーフ | △ | チームワーク・リーダーシップの設問では使えるが、中学時代のエピソードとして印象が薄い。より近い時期の実績（03・05）が優先される |
| 07_課外活動_東海道五十三次 | △ | 継続力・目的ドリブンは伝わるが、経営企画・商品企画の求める資質（折衝力・ビジネス視点）との接続が弱い。単独では使いにくい |
| 17_自己分析_OS2.0刷新 | △ | コミュニケーション哲学・折衝への意識改善は経営企画が重視する折衝力と方向性が合う。ただし実績ベースのエピソードではないため、面接での深掘りに耐えられるか慎重に判断する必要がある |

### 現在のエピソード群でカバーしにくい観点

| 不足している観点 | 説明 |
|----------------|------|
| 新商品・新サービスを0から企画した経験 | 商品企画開発の核心。「あるものを改善する」エピソードはあるが「ゼロベースで価値を設計する」エピソードが現時点では見当たらない |
| 多様なステークホルダーとの折衝・調整経験 | 経営企画で必須の能力。インターンの上司対応はあるが、対等な立場での社外折衝・交渉エピソードはない |
| 顧客・市場インサイトから商品・施策を発案した経験 | 商品企画に直結。何かの課題を起点に「新たな顧客価値の設計」まで行ったエピソードがあると強い |

→ 追加できるエピソードの方向性: ゼミや個人活動の中で「誰かの課題を聞き、解決策を提案・実行した」「何か新しい取り組みを起案してチームに展開した」経験があれば補完に使える可能性があります。

## 補足: 志望整合性の注意点

石田さんの現在の志望軸（資産運用・リサーチ・インサイトで意思決定を変える）と、ビューカードの経営企画・商品企画職との間には方向性の差があります。

- ビューカードは「クレジットカード会社の経営企画」であり、扱うデータ・意思決定の対象は**消費者行動・カード取扱高・提携先交渉**が中心
- 自己分析（01・12）に記載の「やりたくない仕事」（BtoC低単価・意味のない反復・DXの手前に長く張り付く）との矛盾が生じる可能性がある
- **志望動機でこの矛盾を丁寧に解消する**（なぜ資産運用でなくビューカードの経営企画なのか）説明が必要
