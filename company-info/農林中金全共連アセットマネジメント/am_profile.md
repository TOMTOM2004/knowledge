## C群: スチュワードシップ

### C-1 Stewardship integration ◎
- ESG評価→目標株価プレミアム/ディスカウント反映
- **ESGスコア4.5以上で原則投資不可**（ソブリン・クレジット両方）
- 株式: ESG評価は投資判断の重要要素
- 債券: **独自スコア1.0-5.0 + NGFSシナリオ**、信用力評価の1項目としてESG反映
- **PRI 2024年次評価 8項目中5項目 5つ星**
- 2024/3 NZAMI参画、2030年中間目標: 運用資産75%対象にカーボンフットプリント50%削減

### C-2 議決権行使方針・実績 ◎
- **2025/5-6月株主総会**
  - 会社提案13,177件、**反対率6.5%（他社比最低）**
  - 取締役反対率6.2% / 監査役12.1% / 会計監査人12.5%
  - 役員報酬3.4% / 退任慰労金92.6%
  - 剰余金処分0.1% / 組織再編29.4% / **買収防衛策100%**
  - 株主提案308件、**賛成率19.5%**（79社対象）
- **2024年1-12月**: 会社提案21,704件、反対率8.8%、株主提案365件、賛成率25.2%
- **ISS + グラスルイス両方利用を明記**（他AMと異なる独自ポジション）
- ISS議決権行使事務代行サービスも利用
- 反対会社提案の理由: 構成（社外性・女性）、政策保有株、独立性確保未成、計算書類監査未了、ストックオプション希薄化

### C-3 対話テーマ・実績 ◎
- **2024年度 合計580件**
  - 株式アクティブ 184件 + 株式パッシブ 76件 + 債券 320件
- 手法別: 直接対話299 + 外部委託279 + 協働2
- **テーマ別実施 1,715件**:
  - 経営戦略346件
  - ガバナンス244件
  - 財務戦略196件
  - 社会159件
  - 環境116件
  - 情報開示（ESG）118件 + 情報開示（経営戦略等）112件
- 気候変動244件、人権・多様性116件、自然資本52件

### C-4 Stewardship autonomy ◎
- **独自NZAM ESGスコア**（独自ビュー構築、マテリアリティマップ）
- **NGFSシナリオ下の独自ソブリンスコア**（1.0-5.0）
- **クレジット独自スコア**（1.0-5.0、4.5以上は投資不可）
- 協働エンゲージメント 2件
- 議決権行使: ISS/グラスルイスの助言は「当社方針に基づく助言」要求し、最終判断は自社
