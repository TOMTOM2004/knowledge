## 【調査結果サマリー（7軸）】

### ① 公式情報
- **日本唯一の国際カードブランドホルダー**：VISA/Mastercardが「ブランドのみ」なのに対し、JCBはブランドホルダー＋イシュア（カード発行）＋アクワイアラ（加盟店管理）の**3役を同時に担う**独自構造
- **募集コース**：総合職群G職（オープン採用・IT/デジタルアサイン）、習熟職群S職
- **中期経営計画「Plan 2028」**（FY2025〜）：既存事業強化・構造改革・基幹システム「JENIUS」刷新・BPR/デジタル化推進
- **求める人材（公式）**：「現状に甘んじず自ら踏み出せる人」「個の力を周囲のために発揮できる人」「失敗に臆せず前を向ける人」

### ② 求める人材像（公式+体験談照合）

| 軸 | 公式表現 | 行動レベルに落とすと |
|---|---------|-------------------|
| チャレンジ精神 | 「現状に甘んじず一歩踏み出す」 | 過去に誰もやっていなかった取り組みをした・失敗しても再挑戦した |
| 利他性・協働力 | 「個の力を組織のために発揮」 | 自分の強みをチームの弱点補完に使った具体経験 |
| 粘り強さ・失敗耐性 | 「失敗に臆せず貪欲に前向き」 | 失敗から何を変えたかが言える |
| パーソナリティ重視 | 体育会系優遇なし。人物本位 | 想いと行動の一貫性が見られる |

- **合格者傾向（体験談）**：話の振り方・場の協調性を評価（GD「話題振りで合格確信」体験記あり）。面接は柔らかく穏やかな雰囲気。深掘りより人柄確認フェーズ

### ③ 選考実態（体験談）

| 項目 | 内容 |
|-----|-----|
| GDの形式 | オンライン・学生5〜6名・面接官1名・30分 |
| GDの評価軸 | 協調性・場のコントロール・話の振り方 |
| 1次面接の長さ | 約15分（短い） |
| 面接官 | 人事または現場中堅社員 |
| 雰囲気 | 和やか・穏やか・笑顔で反応してくれる |
| 質問傾向 | 自己紹介・ガクチカ・ES深掘り |

> 【要確認】「グループ面接（複数学生を同時面接する形式）」か「GD」かは体験記でも表記揺れあり。GD通過後の個人1次面接の可能性もある

### ④ 抽象語の具体化
- **「チャレンジ」の実態**：新規事業・海外展開（東南アジア等）・JENIUS刷新など大規模システム変革に若手が関わるケースが想定される。「挑戦できる環境」は新しい決済技術（タッチ決済・QUICPay）の展開も含む
- **「DX/デジタル化」の実態**：基幹システムJENIUSの全面刷新中。SaaS/クラウド活用。IT/デジタルアサイン採用コースは実際に開発側に配属

### ⑤ 現場業務
- **若手G職の実態**：ジョブローテーションあり。マーケティング・加盟店開拓・海外事業・リスク管理・IT等多様な部署を経験
- **IT/デジタルアサイン採用**：エンジニア寄りの配属（JENIUS刷新プロジェクト等）
- **1〜3年目**：オープン採用は配属先未定。具体業務は入社後に決まる【要確認】

### ⑥ 業界前段（なぜ決済/カード業界か）
- **銀行/証券との差分**：JCBは「お金を運ぶ・増やす」のではなく「**お金が動く瞬間（トランザクション）のインフラ設計**」が業務の本質
- **社会的役割**：キャッシュレス化・金融包摂・訪日観光客へのサービス品質向上
- **隣接業界との差別化**：Visa/Mastercardが外資ブランドのみなのに対し、JCBは日本発＋3機能統合で「全バリューチェーンを内製できる唯一の国内企業」
- 【要補強】「なぜ保険・銀行ではなくカード/決済か」の前段は薄い。自分の志望軸（意思決定支援・DS活用）とのつながりを言語化要

### ⑦ 中計・IR差分分析
- **推進中の重点施策**：Plan 2028（BPR/デジタル化、JENIUS刷新、海外アライアンス拡大）
- **現状の達成度**：JENIUS刷新は進行中だが大規模基幹システムのため課題あり（enterprisezine報道）。具体的なKPI・達成率は非公開企業のため数値不明【根拠不明】
- **できていないこと（ギャップ）**：JENIUS刷新に伴う現場の移行コスト・開発リスクが課題として報じられている
- **面接活用**：「JENIUS刷新で生まれる業務変革に自分の仕組み化・自動化スキルをどう活かすか」→ 逆質問・入社後にやりたいことに接続可能

### カテゴリB: 志望動機系

---

**Q3: なぜJCBを志望するのですか（なぜこの業界か含む）**

```
回答骨格:
- 業界前段（なぜ決済か）:
  - 「お金を運ぶ・増やす」業務ではなく「お金が動く瞬間のインフラを設計する」
    という役割に惹かれた
  - 1トランザクションの背後に積み上がるデータが、
    消費行動・社会動向の解像度を上げる素材になると考えている
- なぜJCBか（競合差別化）:
  - VISA/Mastercardはブランドのみ。JCBは「ブランド+発行+加盟店管理」の
    3機能を持つ唯一の日本発国際ブランド
  - 全バリューチェーンを内製できるため、決済の仕組み全体を設計できる
  - JENIUS刷新・BPRで「仕組みそのものを変える段階」に踏み込んでいる
- 自分との接続:
  - 仕組みを設計する強みを、決済インフラという社会インフラ規模で使いたい

自己接続チェック:
- 価値観「設計者として構造を変える」とインフラ設計の親和性 ○
- 危険ワード: 「社会貢献」「グローバル」単体で言わない。具体的な機能差分で語る ◎

深掘り問答:
- 深Q1: 「それはVISA/Mastercardの日本法人でも言えませんか？」
  → VISA/Mastercardは外資ブランドのみ提供者。イシュアもアクワイアラも
    機能しない。JCBは3機能を統合するため、
    決済プロセス全体の設計に関われる唯一の選択肢
- 深Q2: 「なぜ他の決済会社（PayPay・楽天Edy等）ではないのか？」
  → QRコード決済は国内特化型。JCBは国際ブランドとして
    世界150以上の国・地域で使われており、
    グローバルな決済ネットワーク設計に携わりたいという軸がある

反論耐性:
- 「それは建前では？」への対応:
  → インターンで仕組みを変えた経験が原点。
    「誰かがやらなければならない設計の仕事」を担いたいという
    行動パターンが一貫している
```

> 【要補強】「なぜ銀行・証券・保険ではなくカード/決済か」の前段が薄い。自分の「インサイト・意思決定支援」志向とカード会社の業務性質のつながりを事前に言語化しておく。

## 【GD対策（グループ面接がGD形式の場合）】

```
JCBのGD傾向:
- 形式: オンライン・5〜6名・30分
- 評価軸: 協調性・話の振り方・場への貢献度
  （「話題振りで合格確信」という体験記タイトルから協調性評価が強い）

GDで取るべき役割:
- 序盤: 論点の整理役（「今回の問いを〇〇と解釈しますが合ってますか？」）
- 中盤: 発言していない人へのパス（「〇〇さんはどう思いますか？」）
- 終盤: まとめ役（「全体を整理すると〜という結論が出ました」）

やってはいけないこと:
- 発言量を稼ごうとして場を支配する
- 議論と関係ない発言をして時間を浪費する
- 論破・対立路線（JCBは協調性評価が重い）

自分の強みを活かすなら:
- 「構造化」スキルを使って論点を整理するファシリテーター役が最も合っている
```
