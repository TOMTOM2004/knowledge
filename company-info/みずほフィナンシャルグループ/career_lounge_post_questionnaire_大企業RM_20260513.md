# 聞けてよかった話 ピックアップ（公開情報では取れない一次情報）

中堅中小RMと並列で読める粒度で、大企業RM ならではの解像度を上げる一次情報を整理。

## A. 大企業RM 業務の解像度を上げた話

### A-1. 本店営業部は「ほぼ全員兼職」
- 本店営業部はほぼ全員が銀行・証券の兼職部。福崎氏ご自身も兼職で、みずほ証券の名刺を所持。
- ただし「みずほ証券の証券マンと同じ知見を持って動けるか」というと現実的には厳しく、**実態は「証券マンと一緒に動く」**スタイル。
- → **示唆**: 2009年導入の兼職制度（事前調査済）が、大企業領域では「選抜型」ではなく**「部単位の標準形」**にまで広がっている。広報資料の制度紹介と現場の運用にはギャップあり。

### A-2. 兼職での業績評価設計（クロスインセンティブ）
- 信託・証券で上がった業績評価が**同額そのまま自分の成績にも反映される**設計。
- → 「信託・証券に流したら自分の成績にならない、めんどくさい」というネガティブインセンティブが**制度的に排除**されている。
- → **示唆**: One MIZUHO は「理念」や「物理近接」だけでなく**評価・KPI 層でも担保**されている。中堅中小RMで聞いた「同じビル・同じ看板」（物理）と組み合わせると、3層構造（理念／物理／評価）で連携を語れる。

### A-3. RM の自己定義は「コントロールタワー」
- 「証券／信託／海外取引／外国為替／資金管理を、自分の中ではすべて**同じ部署位置付け**にしている」。
- 「銀行の中なのか外なのかってあんま関係ない」。担当先に必要なら、機能の所属組織を問わず引き出してくる。
- → **示唆**: 中堅中小RMの「コントロールタワー機能」と表現が完全に一致。**規模が変わってもRMの自己定義が共通**であることを確認できた。

### A-4. 中堅中小と大企業の差分は「ステークホルダーが増えるだけ」
- 「根本は変わらない。会社をどうしていきたいかを一緒に議論する」。
- 大企業ではオーナーに加え市場の見方が乗る → ステークホルダーが増える → **打ち手も増える**、という構造。
- → **示唆**: 中堅中小 → 大企業のキャリア接続性が高い。志望動機で「中堅中小から入って大企業に上がっていく」キャリア像を語る際の論拠になる。

## B. 産業調査部との連携の手触り

### B-1. 大企業ほどリサーチ領域に「深く」入る
- 中堅中小では業界動向はトッピング程度、メインは「お客様の取引先（例: トヨタ）の動向」。
- 大企業（鉄3社・非鉄・電力・エネオス・オイル等）では**業界そのものをどう動かすかの議論**が求められる → 産業調査部との連動が必須。
- → **示唆**: 「業界を代表する企業ほど、産業調査経験者がRMを担当する」。資源素材・半導体・自動車セクターは特に顕著。

### B-2. 産業調査レポートは「双方向プロセス」
- 「ヒアリングなしでは産業調査部もレポートを作れない」。
- 日頃のリレーションで吸い上げた一次情報 → 議論 → レポート化、というサイクル。
- 顧客から「このレポートの内容違うかな」という**フィードバックは現実にあり**、産業調査部にとってもありがたい修正源として機能。
- → **示唆**: 公開レポートはアウトプットだけが見えているが、裏では**RM ↔ 顧客 ↔ 産業調査部の三角インタビュー**が回り続けている。

### B-3. 産業調査部の同席・現場見学同行
- 深い議論はRM単独ではなく**産業調査部も同席**することが多い。
- お客様の**工場見学にも産業調査部が同行**するケースあり。
- → **示唆**: One MIZUHO の連携は「商品を一緒に売る」だけでなく「**業界知見を一緒に積む**」現場レベルにある。

### B-4. リサーチ&テクノロジーズ内製化（2026年4月）で足枷が消えた
- 2026年4月にリサーチ&テクノロジーズが内製化（≒合流）。
- それ以前は別会社のため、情報共有同意書がない先には「バンカーズアイデアで仕掛ける」相談ができず**仕掛け系で足枷**だった。
- 内製化後は**動きやすい体制**になり、営業担当が一番効果を実感。
- **内製化の効果は 9→10 型**（0→1ではなく既存連携の摩擦解消）。具体には、**日々の業務でクライアントに情報提供する際の情報共有同意書のハンコ取得作業が不要になった**こと、これがそのまま「仕掛けの初動スピード」に効いている、という業務粒度の手触り。
- → **示唆**: 競合との差別化要素として、**「2026年4月以降の体制刷新」**は時期的に語れる新ネタ。志望動機で「今この瞬間のみずほに入る理由」として使える。
- → **想定問答での使い方**: 「みずほの良いと思う点」を抽象的に語らず、**「同意書のハンコ業務が消えた」**という業務粒度の話に落とすことで、他社志望者との差別化になる。

### B-5. 産業調査部の源流＝日本興業銀行
- 「うちの産業調査部は工業銀行（日本興業銀行）からの源流」。
- 政策銀行として日本の政策をどう動かすかを考えてきた歴史 → コンサル会社の相対的調査とは「訳が違う」強み。
- → **示唆**: 三井住友・三菱UFJの調査部との差別化を「**興銀ルーツ × 政策銀行的視点**」で語れる。

## C. メインバンク／非メイン取引・他行戦略

### C-1. 地域別プレゼンス（東京 vs 名古屋）
- 東京: みずほは**東京都の指定金融機関**でプレゼンス高い。
- 名古屋: 三菱UFJが東海銀行を源流に持ち**三菱先が多数**。みずほは追走側。
- → **示唆**: 「みずほは東京強い」だけでなく、エリアごとに競合構造が違うことを把握。志望動機で「どのエリアでどう戦うか」を語る際の地理感覚になる。

### C-2. 一行取引は「戦略的狙い目」
- 中小企業で借入不要な会社は一行取引にしがち → 取引銀行に**やりたい放題されている**ケースが多い。
- 「三菱しかやってない先」「意外と手数料・金利が高い」 → ここに**戦略的に切り込む**のがメガの一手。
- BCP観点や貸出態度・与信の考え方の違い（メガ vs 地銀）も入り込み口。
- → **示唆**: 「シェアを奪う」ロジックが、単純な金利競争ではなく**情報の非対称性**を突く形で設計されている。

### C-3. 非メインの入り方は「セカンドオピニオン」
- 喩え: 風邪は近所の医者で済むが、がんになったら**セカンドオピニオンを欲しがる**のと同じ。
- メインバンクが全てを言っている／全提案を受けているわけではない → **非メインこそ角度を変えた提案**で食い込める。
- → **示唆**: 非メイン取引で何ができるか、という典型的な深掘り質問（事前調査済）への回答骨格が得られた。

### C-4. メインへの「仁義」
- ただし、メインに**内緒（ダマ）で動くのはお行儀よくない**。
- 不動産売買やM&A案件など実弾レベルでは、メインに何かしらの整理（情報共有）が必要。
- 入口のディスカッション段階でも、いったんメインに相談せざるを得ない局面あり。
- → **示唆**: 「非メインからのアグレッシブな提案」と「業界の暗黙ルール」のバランス感覚が、大企業RMでは制度より重い。

### C-5. 格付け × 引当 × スプレッド金利の仕組み
- 銀行は格付けに応じて引当を積む（100貸して何が返るかの想定が業績に織り込まれる）。
- 10年貸出なら10年で格付け相応の引当分を回収 → **スプレッド金利**として上乗せ。
- 途中で業績悪化局面に金利を上げるのは大変 → **最初からスプレッドを取る**設計。
- → **示唆**: 銀行ビジネスの「貸出単独では稼げない／外為・M&A・不動産との総合採算」というモデルが、引当ロジックから腹落ちした。

## D. キャリア（RM ↔ 産業調査部 ↔ セクター）

### D-1. RM ↔ 産業調査部の往復はキャリア上「多い」
- 本店営業部内にも産業調査部出身者が多数。
- 業界を代表する企業ほど、**産業調査経験者がRMを担当する**傾向。
- → **示唆**: 「RM → 産業調査 → RM」が典型的キャリアトラックの一つ。事前調査の仮説と一致。

### D-2. 産業調査部に移る際は「サプライチェーン上の隣セクター」もあり得る
- 同じセクター継続が理想だが、本人意向次第で別セクターも可能。
- 例: 資源素材RMから飲料メーカーセクター（缶メーカー経由のサプライチェーン隣接）への異動シナリオを口頭で例示いただいた。
- → **示唆**: 資源素材は**川上ポジション**で「川下にはいくらでも行ける」幅広い起点になる。志望動機で「最初に資源素材を選ぶ意義」として語れる。

### D-3. 1年目で社長と会話できるのは「銀行ならでは」
- 23歳で社長アポが入ること自体、銀行以外の業界ではほぼ不可能。
- 「銀行は重視している立場」「話さなきゃいけないと会社が感じてくれている」 → **時間を取ってもらえる構造的優位**。
- → **示唆**: 新卒キャリアの初速の取りやすさを定量・定性両面で語れる素材。

## E. GX案件・サステナブル業務の実態

### E-1. 案件数は「そんなに多くない」
- GX推進機構（経産省外閣団体）と連携してサステナブル企画部が運営。
- **認定取得自体が大変**で、GX案件として走るプロジェクトはそこまで多くない。
- → **示唆**: 広報では大きく見える GX 領域だが、現場の感覚としては**まだ立ち上げ期**。志望動機での過剰な GX 推しは避けるべき。

### E-2. 補助金は「最初に確定」型
- 補助金は認定時点で予算が確定する → 途中で打ち切られるリスクは小さい。
- 例外: **カーボンクレジット系**は認定が外れると金利補給が消え、その分を顧客負担に転嫁する必要あり。
- → **示唆**: 「補助金打ち切りシナリオを織り込んだ議論」（事前用意した質問）に対し、「制度設計上ほぼ織り込み不要、ただしカーボンクレジット系は別」という**具体的境界線**を得られた。

### E-3. 補助金 + 融資のセット型
- プロジェクト全額を補助金で賄うことはなく、補助金部分以外は**銀行融資で組成**するセット型。
- → **示唆**: GX案件の中での銀行の取り分（融資側）が見える。RMがGX案件にどう絡むかの絵が描ける。

## F. 大企業RM のやりがい・スケール

### F-1. 組織で動くダイナミックさ
- 中堅中小は社長に直で刺せる強みがある一方、大企業は「組織で動かなきゃいけない」ことの**自覚と設計**が必要。
- 「なぜみずほを選ぶか／なぜこの提案を経営に上げるか」の理屈を**経営の意思決定構造に合わせて作り込む**。
- → **示唆**: 中堅中小と大企業の「やりがいの質的違い」を、ドライな組織意思決定の話を含めて整理できた。

### F-2. 米国グリーンヒル買収のリソースを使える
- 数千億規模の案件では、買収子会社の米国グリーンヒル（M&Aブティック）まで動員する。
- 中堅中小では使う相手にならない武器が**大企業領域では本気で投入される**。
- → **示唆**: 「みずほのリソースの厚み」を、抽象論ではなく**具体的な子会社名・案件規模**で語れる素材。

### F-3. 産業調査部の使い方が「中堅中小時代と別物」
- 中堅中小RM時代は産業調査部のことを「正直そこまで知らなかった」。
- 大企業RM になると**ほぼ常時絡む**、産業調査部側も「現場で何を言ってたか」を聞きたがる関係に逆転。
- → **示唆**: 同じみずほ社員でも、所属によって産業調査部との距離感が大きく違う。職種選択（中堅中小／大企業／IB）の意味が変わってくる重要観点。
