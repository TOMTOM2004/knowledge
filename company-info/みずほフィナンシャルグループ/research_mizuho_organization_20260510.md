# みずほ銀行 大企業RM部署 組織構造調査

## TL;DR（5行）

1. **物理拠点は「みずほ丸の内タワー」**（千代田区丸の内1丁目、2020年9月竣工、地上29階）。「大手町本店」は慣用表現で**厳密には丸の内**
2. **2021年4月「ナンバー部」を廃止し、業種横断5インダストリーグループ（IG）体制に再編**
3. 5IG: **①自動車・テクノロジー / ②資源・素材 / ③商社・不動産・フィナンシャルスポンサー / ④社会・産業基盤 / ⑤情報通信・リテール**
4. **資源・素材グループ**は第一部・第二部・第三部以上（**第二部に化学・タイヤチーム**が内包されることは実名確認済み）
5. **キャリアラウンジ**: 社員1名×学生1名・40-50分・通常3回・基本オンライン

## 2. 2021年4月再編：5インダストリーグループ体制

### 旧体制（2021年3月まで）

「営業第○部」「法人第○部」のナンバー部による業種別縦割り体制。

### 新体制（2021年4月〜）

| グループ | 確認された傘下部数 |
|---------|-----------------|
| 自動車・テクノロジーグループ | 推定複数部 |
| **資源・素材グループ** | **第一部〜第三部以上** |
| 商社・不動産・フィナンシャルスポンサーグループ | 第一部〜第三部 |
| 社会・産業基盤グループ | **第四部まで確認済み** |
| 情報通信・リテールグループ | 第三部まで確認 |

**目的**（日経2020-12記事より）: 産業界での業態超え提携・買収増加に対応。業種横断の方が顧客の事業構造を反映する。

**実態**: 「業種横断」と銘打つが、各IG内部はさらに業種別チームに細分化。完全な横断運営ではない（外資就活ソースで「資源・素材第二部/化学第二チーム」の存在確認）。

## 3. 資源・素材グループの内部構造

### 確認された事実

- 第一部・第二部・第三部の存在（2026年4月時点）
- 第二部に「化学第二チーム」内包 = 化学・タイヤ・ゴム系を担当（東証プライム上場タイヤ・化学メーカー担当のRMが実名で確認）
- 第三部 = 貴金属・製缶（面接官プロフィールより）

### 第一部・第二部・第三部の使い分け（推定）

公式の業種対応表は非公開。以下は二次情報による推定:

| 部 | 担当業種（推定） | 根拠 |
|----|---------------|------|
| **第一部** | **鉄鋼・非鉄金属（銅・アルミ等）系** ? | 業種規模が最大かつ歴史的に資源・素材の中核 → 第一部に配置の自然性 |
| **第二部** | **化学・ゴム・タイヤ・化学繊維** | 「化学第二チーム」の存在確認 |
| **第三部** | **貴金属・製缶・紙パ・ガラス・セメント等の残余** | プロフィールで貴金属・製缶確認 + 業種規模が小さい残余セクター集約と推定 |

### 第三部に貴金属・製缶が同居する理由（推定）

1. **素材加工バリューチェーンの連関**: 貴金属（素材） + 製缶（金属加工川下）= 「金属素材の加工・応用」大括り
2. **業種規模**: 鉄鋼・化学のような大規模業種ではないため、残余を集約
3. **歴史的経緯**: 旧ナンバー部時代の担当割り振りの名残

### 資源・素材グループ全体のカバー業種

鉄鋼 / 非鉄金属（銅・アルミ等）/ 貴金属 / 化学 / ゴム / タイヤ / 化学繊維 / 紙パルプ / ガラス / セメント / 容器包材（製缶・PETボトル）/ 素材系商社（阪和興業等）

→ **みずほ産業調査の「素材産業」カバー範囲と概ね一致**

## 4. 「5社グループ」の意味

「5社/G」= 1人のRMが担当する **大企業グループの数**。

- 中堅中小RM: 30社/G（実発言）= 1社あたり関与は浅め
- 大企業RM: **5社/G** = 1社あたり関与は **6倍密度**
- グループ会社を束ねて「○社グループ」として管理

**プロフィール解釈**:「貴金属・製缶メーカー、5社グループ」= 例えば「住友金属鉱山グループ + 三菱マテリアルグループ + 東洋製罐HDグループ + 大和製罐グループ + ホッカンHDグループ」のような5つの大企業グループを担当している可能性
