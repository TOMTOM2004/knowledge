# 志望動機仮説（RM用）— 2026-04-18

ステータス: 仮説段階。SC用を別途構築予定。

## 核心構造

### やりたいこと
中堅中小企業のRMとして、定量分析をAIで自動化し、浮いた時間で**フィジカルなアクション**に注力する。

- 融資先の工場を見に行く
- 経営者・従業員に直接会って話を聞く
- 業界の展示会に足を運び、どういう言葉が刺さっているか・現場の温度感を掴む
- 取引先にヒアリングし、受注パイプラインの実在性を検証する

これにより、財務諸表だけでは見えない「事業の成長性」「マーケットの需要」「現場の温度感」を与信判断や提案に反映させる。

### なぜみずほか
- **UPSIDERを連結子会社化（460億円）した唯一のメガバンク**。法人カード利用社数8万社超・AI自動審査で従来1ヶ月→平均10営業日に短縮。BLUE DREAM Fund（ベンチャーデット）累計243億円・最短1週間融資実行。MUFGのSakana AI連携はパイロット段階、SMBCのTrunkはAI与信モデルの内製化には至っていない。「定量自動化のインフラ」が実装済みなのはみずほだけ
- **産業調査部が銀行本体に内蔵**（他メガはシンクタンク外出し）。RMと共に取引先企業に直接入って経営課題分析→戦略提案→金融ビジネスへの橋渡しを行う。公式採用ページで「みずほにしかない機能」と明記。MUFGの三菱UFJリサーチ&コンサルティング、SMFGの日本総合研究所はレポート提供が中心で、RM協働の度合いが異なる
- **銀行×証券×信託の三位一体を完全自前で保有**: MUFGは証券がモルスタ合弁（60:40）でFW違反処分歴あり、SMFGはグループ内に本格的信託銀行がない（三井住友信託は別グループ・別上場）。完全自前で三法人＋統一人事「かなで」を持つのはみずほだけ
- この3つが揃っているから、「定量はAI・定性は人間」の分業を中堅中小RMとして最も早く実現できる環境

### なぜ自分か
- AIを「使う側」ではなく、**自分でPoCを作って回し、効果を実証してからIT部門に引き継いで全社展開する**人材
- 個人の課題解決 → ナレッジ化 → 全員が享受できる世界を目指す
- 実例: SLMでエラーパターンを蓄積・プロジェクト横断で横展開。Claude Codeを使ったTodo管理システム（修正項目の一覧化・原因特定・タスク分解の業務フロー自動化）をPoCとして構築・運用中

## コアテーゼ
「定量分析はAIで自動化すべき → 浮いたリソースで人間にしかできない定性判断（生の声・ニーズの真偽・成長性の肌感覚）に注力する → そこが価値創造の本丸」

- AMアナリストの例: 調査時間が浮いた → エンゲージメント（議決権行使・CEO面談）に注力 → 社会還元・株価向上の次のステップへ
- RMの例: 稟議作成時間が浮いた → 工場訪問・従業員ヒアリング・展示会参加に時間を使える → 財務諸表に表れない事業の本質を掴む

## 想定される反論と対策

1. **「浮いた時間は新規開拓に使えと言われたら？」**
   → 新規開拓こそ定性判断の塊。AIがリスト作成を自動化し、人間が「この企業に今アプローチすべきか」を判断する

2. **「AIを作るのはIT部門では？」**
   → IT部門に実装させるのではなく、自分がAIを使って自分でPoCを回す。個人で回ったものをIT部門に持ち込んで全員が使えるツールにする。これが最もスピード感のある実装

3. **「SLMの話はエンジニアリング寄りでは？」**
   → Todo管理システムの例で補強。現場の社員が既にやっていた業務（修正管理・原因特定）をテクノロジーで業務フロー自体を変えた。本質は「既存業務の再設計」

4. **「承継なら銀信一体のりそなの方が速いのでは？」**
   → りそなには証券子会社がない。事業承継で信託を使った後、M&A・IPO・社債発行は全て外部依存。「入口」は強いが「出口」がない。みずほは承継の先の成長戦略（海外進出・M&A・IPO）まで途切れなく伴走できる。加えてみずほ信託銀行は手数料収入比率が大手信託銀行中最高で信託特化度が最も高い。プラネットブース100拠点超＋統一人事＋オプトアウト方式で、別法人のデメリットは制度とインフラで克服済み

5. **「中堅中小が好きなら地銀の方が地域密着できるのでは？」**
   → 地銀FGは証券（M&A・IPO・DCM）、信託（承継・不動産）、海外ネットワークが外部依存。企業が成長してM&AやIPOを目指す段階になるとメガに案件が流れる構造がある。産業調査もコンコルディアは神奈川経済、ふくおかは九州経済が中心で、みずほ産業調査部の全産業×全地域カバレッジとは質的に異なる。UPSIDER/BLUE DREAM Fundの全国カバレッジも地銀のVC（地域限定的）とは別次元。「最初から成長の全ステージに伴走できるメガのRMでありたい」

6. **「MUFGの方が規模も大きいのになぜみずほ？」**
   → MUFGは大企業・超大型案件に特化。中堅中小RMとしての提案の幅はみずほが上。UPSIDER/BLUE DREAM Fund（デット型で最短1週間）、産業調査部のRM協働、プラネットブース100拠点超は全てMUFGにない機能。さらにMUFGは各社独自人事制度で分断、証券がモルスタ合弁でFW違反処分歴あり。「規模ではMUFGだが、中堅中小への提案の幅と一体運営のガバナンスではみずほが上」
