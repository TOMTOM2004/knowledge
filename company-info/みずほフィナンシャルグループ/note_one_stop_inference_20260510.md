**みずほの優位性の核心**:
1. **4機能（銀行・信託・証券・シンクタンク）を完全自前**で持つ唯一のグループ
2. **2026年4月のみずほR&T銀行統合**で、シンクタンクが銀行内に組み込まれ動線がさらに短縮
3. 旧日本興業銀行のDNA（大企業・産調・長期伴走）が資源・素材セクターで特に強い

## 4. ワンストップが「効かない / 機能しない」領域（推論）

| 領域 | 理由 |
|------|------|
| 商社機能（自前トレーディング） | 銀行は商社にはなれない。三井物産・三菱商事・伊藤忠が領域所有 |
| 海外現地での執行 | 海外パートナー（Lazard・Greenhill等）に任せる領域あり |
| 製造業の業界知識深掘り | 産調・R&Tがあっても顧客自身の知識を超えない |
| 法律・会計の専門業務 | 監査法人・弁護士事務所への外部委託 |
