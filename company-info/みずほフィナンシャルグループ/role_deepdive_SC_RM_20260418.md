# SC・RM 業務実態 Deep Dive（2026-04-18）

志望動機構築の土台として、SC/RMの「現場で何をしているか」を具体化した調査。

## Part 1: みずほ証券 SC（リテール営業）

### 1日の業務フロー

| 時間帯 | 業務内容 |
|---|---|
| ~7:30 | 出社。モーサテ・日経で海外市場・為替確認 |
| 8:00 | 投資情報部レポート確認 |
| 8:20 | 朝会: 本社からの相場見通し、課長から注力商品・推奨銘柄、前日成績振り返り |
| 9:00-11:30 | 東証開場。電話注文受注、既存顧客への相場連絡・提案電話 |
| 11:30-12:30 | 前場終了。午後の訪問準備、提案資料作成 |
| 13:00-15:00 | 後場。電話対応＋投信・募集商品の提案アポ取り |
| 15:00-18:00 | **外出訪問が本格化**（既存フォロー・新規提案・ヒアリング） |
| 18:00-19:00 | 帰社。営業日報・CRM入力、翌日準備 |

### 案件の流れ

1. **新規開拓**: グループ銀行（みずほ銀行）からの紹介が主要チャネル。飛び込みは新人のスキル訓練的位置づけ
2. **ニーズ発掘**: 初回面談で年齢・収入・家族・投資経験・意向を「傾聴」。商品を売るのではなく課題を聞く
3. **提案**: 資産運用ポートフォリオ作成（株式・債券・投信・ラップ・NISA等）。グループ連携で融資・信託・不動産も含む総合提案が可能
4. **受注**: 適合性原則の確認必須
5. **アフターフォロー**: 月1回程度の定期訪問、相場急変時のタイムリー情報提供、ライフイベント対応

### 顧客接点の実態

- **担当顧客数**: 約300件（ウェルスマネジメント課の事例）
- **訪問頻度**: 月1回程度（重要顧客はより高頻度）
- **接触手段**: 対面・電話・オンライン（電話・オンライン専門を2025年度に100人体制へ倍増方針）
- **新規vs既存**: 既存フォロー中心。新規はグループ銀行紹介が主チャネル

### 銀証連携の実務

- **兼職体制**: みずほ銀行の大企業法人RMは邦銀で唯一みずほ証券と兼職
- **プラネットブース（128箇所）**: 銀行ロビー内にみずほ証券の正規店舗。銀行来店客にその場で証券サービス提供
  - 実例: 銀行では対応困難な「カナダドル小切手の送金」を銀行部門と連携解決
- **インダストリーグループ**: 業種別専門チームがクロスセクター案件を捕捉

### GIB（投資銀行部門）ジュニアの実務

- **DCM**: 日本企業の外貨建て債券・海外企業の円建て債券の引受。起債タイミング・年限・発行額の提案。シンジケーション調整
- **ECM**: 株式発行による資金調達の立案・執行。業界分析、業績分析、資本政策分析。IPO支援
- **M&Aアドバイザリー**: DD、バリュエーション、ドキュメンテーション。商社・PEファンド・中堅案件中心
- **共通**: リサーチ・データ収集分析・資料作成が主。複数案件並行。税務・会計・法務+英語力

### みずほ証券SC固有の特性

| 観点 | みずほ証券 | 独立系（野村・大和） |
|---|---|---|
| 新規開拓 | グループ銀行紹介が安定供給 | 自力開拓の比重大 |
| 提案範囲 | 銀信証一体で融資・信託も | 証券商品中心 |
| キャリアパス | グループ内異動可能 | 証券業務内 |
| 営業文化 | **個人収益目標を廃止**。「売る→預かる・守る」へ転換 | 収益目標が明確 |

## Part 2: みずほ銀行 中堅中小RM

### 1日の業務フロー

| 時間帯 | 業務内容 |
|---|---|
| ~8:00 | 出社。日経・モーサテで経済動向確認。担当先HPチェック |
| 8:30-9:00 | 朝礼・メール・やることリスト・訪問準備 |
| 9:00-12:00 | **外回り（訪問）**。午前に集中。多い日で5社 |
| 12:00-13:00 | 昼食 |
| 13:00-16:00 | 追加訪問 or デスクワーク（訪問情報整理・財務分析・電話約20件・メール約50通/日） |
| 16:00-17:00 | 内部会議・上司への案件報告・本部との方針相談 |
| 17:00-19:00 | **稟議書作成**・会議資料・翌日準備 |

### 担当・訪問の実態

- **担当社数**: 20~40社（支店・年次で差あり）
- **1日の訪問**: 2~5社
- **訪問頻度**: 重要先は週1-2回、通常先は月1-2回

### 融資案件のフロー

1. **ニーズ発掘**: 訪問時の経営者ヒアリング（設備投資・運転資金・M&A資金等）。決算書・試算表の定期入手→財務分析から先回り提案
2. **稟議書作成**: 会社概要、財務分析（BS/PL/CF）、**資金使途（要資）**、**返済方法（返資）**、担保・保証条件、銀行側メリット、リスク評価
3. **稟議回付**: 担当RM → 係長 → 課長 → 副支店長/支店長。権限超過は本部（審査部）決裁
4. **融資実行**: 契約書類締結、担保設定。申込→実行まで1-2ヶ月
5. **モニタリング**: 定期的な決算書入手（年1-2回）、試算表（四半期）。業績悪化先はウォッチリスト化

### 事業承継案件のフロー（玉谷RMの実例）

1. 担当先の社長・専務の退任意向をヒアリングで把握
2. M&A（売却）を検討 → 次世代役員が「経営権を自分たちで持ちたい」意向を発掘
3. **MBOスキーム**を提案
4. **One MIZUHO**: 銀行（融資）・信託（株式承継信託・相続対策）・証券（株式評価・FA）の関係各所とスケジュール調整
5. **3ヶ月**でゴールから逆算、グループ力総動員
6. MBO成立

### One MIZUHOの実務

- **産業調査部**: 業界別構造分析・将来予測レポート「みずほ産業調査」を発行。RMが訪問前に読み込み、提案根拠に活用。**リサーチャーがRM訪問に同行**するケースもある
- **2026年4月組織変更**: みずほR&Tがみずほ銀行に合併。「みずほ総合研究所」としてRMとの距離がさらに縮まった
- **証券への橋渡し**: RMがニーズ把握 → 銀証連携担当部署 → みずほ証券担当者を紹介 → 共同提案
- **信託**: 事業承継信託（自社株承継）、遺言信託、不動産信託、年金受託

### UPSIDER連携

- 2025年11月より法人顧客にUPSIDERカードの紹介開始
- 与信限度額最大10億円、還元率1.0%、会計ソフト自動連携
- RMが担当先の経理効率化ニーズに対して提案
- スタートアップ向けデットファンドも設立（2023年11月）

### 新規開拓（メガバンク共通）

- 既存顧客からの紹介が最主要ルート
- テレアポ（未取引先リスト）、セミナー・イベント、ビジネスマッチング
- **既存顧客のルート営業が中心**

### みずほRM固有の特性

- 銀信証一体運営の深さがメガ3行で最も進んでいる
- 産業調査部のリサーチャーが営業支援する体制は他行より充実
- みずほ信託: 2025年度から法人/個人の垣根を取り払い約2,000人「コンサルティング部門」に再編
- 事業承継: 約1万6,000社を積極支援する体制（日経2024年報道）
