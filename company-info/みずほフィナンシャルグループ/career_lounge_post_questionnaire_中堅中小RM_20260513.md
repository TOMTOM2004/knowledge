# Career Lounge 事後アンケート 回答（中堅中小企業RM）

## 設問1
> 面談を通じて新たに知ったこと・印象に残ったことを教えてください。

**回答（草案）:**

中堅中小企業RMの仕事が、担当先の属性（オーナー企業／在日外資／ベンチャー）によって意思決定の入り方が大きく変わることを伺えた点が最も印象に残りました。オーナー企業では「社長や財務部長が決めれば会社が動く」ため自分の提案が直接刺さる一方、在日外資では決済権者が日本にいない・本社判断が読めず提案の浸透先が見えにくい、ベンチャーは意思決定が速い反面赤字先行で公庫・保証協会と組む必要がある——という違いを、担当エリアの実例（青山＝化粧品・韓国系イノベ／大手町＝公庫）込みでお話しいただき、RMの「提案ロジックが顧客タイプごとに違う」という解像度が一気に上がりました。

また、銀証信連携が「同じビルに銀行・信託・証券の看板が並ぶ」という物理的設計から始まっており、青山法人部に信託・証券のブースが置かれ、青山の不動産案件（借地権付き売却・BS影響シミュレーション）で信託と月1回ペースで同行訪問されているという具体描写も新鮮でした。連携を制度で語るのではなく、距離・看板・同行頻度といった現場の手触りでお話しいただいたことが、One MIZUHO を自分の言葉で理解する上で大きな手がかりになりました。

---

## 設問2
> みずほグループや本職種に対する理解・興味はどのように変化しましたか。

**回答（草案）:**

事前は「銀行・信託・証券を横断できるグループ総合力」という抽象的な理解にとどまっていましたが、面談を経て、RMが**コントロールタワー**として本部・グループ会社の提案内容を「こういう目的で行くからこの話はしないで」とコントロールしている、というRMの位置取りが具体的にイメージできるようになり、志望度が明確に上がりました。

特に、二家店目になった瞬間に「全部できるやつ」として扱われ、本部連携の判断・後輩教育・難度の高い案件引き継ぎが一気に来るというキャリアの段階設計を伺えたことで、一家店目で場数を積みながら本部の地図を頭に入れる、という最初の3〜4年の過ごし方が逆算できました。「銀行内で証券に異動しようと思えば普通に異動できる」と、グループオープン型の制度面ではなく現場の運用として言及いただいた点も、進路選択の確信材料になりました。

## 設問3
> 今後の選考や個別面談で確認したいことがあればお書きください。

**回答（草案）:**

- **中堅中小RMから他職種・他社への展開**: 中堅中小RMで身につく「コントロールタワーとしての提案統合力」が、後に首都圏法人部・大企業RM・IB・信託・証券へ移った際にどのような形で活きるのか。逆に、最初に大企業RMから入る場合と比較したときの強み・弱みの違い。
- **AI/DX活用の最前線**: MINORI 基幹系に触れない領域でのAI導入（WithChat・面記くん・スライドジェネレーター）がどこまで現場業務を再設計しているか。今後RMが「機能を呼び出す側」と「機能を作る側」のどちらに比重を移していくのか。
- **本部への提言経路**: 意見箱・アンケート・本部連携メッセージ・本部からの情報要求と多経路あるとのことでしたが、若手RMの声が実際にプロダクトや業務フロー改善につながった事例を、もう少し具体的に伺いたいです。
- **承継・M&A への踏み込み方**: 「関係値ない人にいきなり売却話はマナー違反、銀行は決済組織」というスタンスを伺えたうえで、関係構築から承継議論に至る入り口の引き方を、RMと証券・信託がどう分担しているか。

## 設問4
> 面談全体の感想を自由にお書きください。

**回答（草案）:**

公開情報や説明会では到底届かない、「現場でRMが何を考えどう動いているか」の手触りが伝わる面談でした。特に、3メガを横並びで見ると外からはほぼ違いが見えないなかで、「どの会社の社員と一番一緒に働きたいかが一番大事」というアドバイスと、面談相手ご自身が韓国文化・コスメといった趣味を仕事に重ねながら担当エリアを楽しんでおられた姿が重なり、御行の社員の方々と働きたいという気持ちが強くなりました。今回伺った内容を踏まえて、銀証信連携と本部リソースの活用設計について自分なりの仮説を持ち、次の選考に臨みたいと思います。

## A. RM 業務の解像度を上げた話

### A-1. 担当先3類型と提案ロジックの違い
| 類型 | 意思決定の特徴 | RM の打ち手 |
|------|----------------|-------------|
| オーナー中小企業 | 社長／財務部長で確定。提案がダイレクトに刺さる | 部長レベルで完結。自分の力の範囲で全部終わる |
| 在日外資 | 本社判断・日本に決裁権なし・言語・在席の問題 | 意思決定構造の解読そのものがミッション |
| ベンチャー | 速いが赤字先行・常に資金不足 | 公庫協調融資、保証協会付き融資の組成 |

→ **示唆**: 「中堅中小RM ＝ 単一の仕事」ではなく顧客タイプ別に提案ロジックを切り替える職種。志望動機・逆質問で類型別の話に踏み込める。

### A-2. RM のコントロールタワー機能
- お客さんと最も近い窓口として、本部・グループ会社の提案を**統合・取捨**する。
- 「この目的で同行するから、こういう話はしないで」と本部にディレクションを出す立場。
- → **示唆**: RM は「商品を売る人」ではなく「複数機能の交通整理をする人」。志望動機の中核に据えられる。

### A-3. 二家店目で一気に来る役割転換
- 一家店目（〜3〜4年）: 場数・銀行ビジネスの感覚を掴む段階。
- 二家店目: 本部連携の判断を自分で下す／後輩教育／難度の高い案件引き継ぎが「全部できるやつ」として一気に乗る。
- → **示唆**: キャリアの段階設計が明確。最初の3〜4年の過ごし方を逆算できる。

## B. One MIZUHO（銀証信連携）の手触り

### B-1. 物理的近接が心理的距離を縮める設計
- みずほ銀行の建物外壁に「みずほ銀行 信託 証券」と並列表記。他行は別看板。
- 青山法人部の中に信託・証券のブースが置かれている（本拠は隣駅・渋谷）。
- → **示唆**: 連携は制度・看板・距離の3層で設計されている。逆質問で「他行と物理設計が違う理由・効果」を聞ける。

### B-2. 信託連携の典型ケース（青山の不動産案件）
- 課題: 地価高騰で古いビルの建替不能。底地は手放したくないが資金もない。
- 解: 借地権付き建物を売却→デベ・一般法人が建替活用。
- 信託の役割:
  - 建替費用・建替後利回り・売却価格の試算
  - 不動産の値洗いによるBS影響＝相続税シミュレーション
  - 情報共有同意書ベースで月1同行
- → **示唆**: One MIZUHO の具体例として最強の素材。志望動機・面接エピソードに転用可能。

### B-3. 法個連携・地名法人部 vs 支店
- 都心は法人部・支店が分離、地方は一体運営。
- 法人部部長 ＝ 支店長 フラット関係。
- 社長個人運用は支店連携、事務オペは窓口に依頼。
- → **示唆**: 2019年組織再編後の「分離設計」の意味を理解できた。中堅中小RM志望理由の補強材料。

## C. みずほの本部・グループ機能の活用

### C-1. 本部への提言経路は4チャネル並走
1. 意見箱
2. 定期アンケート
3. 連携部署宛のメッセージ
4. 本部からの情報要求（業界・テーマ別）

- 反映実績: 「こういう声を受けたのでこういうシステムを作りました」と本部からフィードバックが来る。
- 改善例: たらい回し問題 → Excel整理 → 特定アドレス送付 → 部署内振り分けの仕組み導入。
- → **示唆**: 「現場の声がプロダクトに変わる経路」を具体例で語れる。

### C-2. 産業調査部の使い方
- 大企業担当は同行可。中堅中小RMはリソース制約で同行なしで「読み物」として活用。
- お客様に紙で渡す（銀行は紙社会）。
- 旧興銀のリサーチ系エリート組織がベースで、レポート品質は業界横断で強み。
- → **示唆**: 単独ではなく「RMの土台」として産業調査を使う発想。

### C-3. ビジネスマッチングの三層構造
1. RM個人レベル: 担当先同士をマッチング
2. 組織レベル: 掲示板にニーズ登録 → グループ内で接合
3. 本部レベル: 有償契約スキーム（キックバック）
- 具体例: トレーダム × みずほ本部の協業（RMが新規開拓→本部が事業化）
- → **示唆**: RM ＝ 単なる融資窓口ではなく「ビジネス起点」の起源になりうる。

## D. AI/DX と MINORI 制約の「線引き」

### D-1. 「言語道断」と「絶対やれ」の二項対立
- MINORI（基幹系）: AI 導入「言語道断」。コスト・障害リスクで触れない。
- MINORI 外: 経営層「もう絶対やれ」「使うことが大前提」。
- → **示唆**: 「銀行のAI＝遅い」ではなく、線引きが明確で**MINORI外は前のめり**という認識転換。

### D-2. 現役で使われているツール

### D-3. RM が欲しいAI機能の方向性

- → **示唆**: ユーザー（＝RM自身）視点でのAIニーズを語れる。「DXを傍観する人」ではなく「使いこなす人」のスタンスが出せる。

## E. 新規開拓・関係構築の実務

### E-1. 新規開拓は部の足元次第
- 全社号令ではなく**部長判断**で2〜3年でウエイトが変わる。
- 青山＝マーケットの新陳代謝が激しく新規前提／大手町＝既存パイプライン創出フェーズ。
- 新規方法: 問い合わせフォームから「御社が気になりました」メール／コスメ裏面の社名へ送付。
- → **示唆**: みずほ銀行の看板で開拓が成立する強みを実例で理解。

### E-2. 関係構築は「自己開示」が起点
- 「自分に興味がないやつに打ち明けたくない」「弱みを見せられない社長に雑談相手として座る」。

- 承継話は重い→関係値ない人にいきなり「会社売りませんか」はマナー違反、MA仲介の仕事。
- → **示唆**: 「銀行は決済組織。関係構築の上で承継話に至る」という順序感覚。ガクチカ・面接で「関係構築の解像度」として転用可能。
