# 貴金属・製缶業界 + 経営戦略議論の質的差

## TL;DR（5行）

1. **貴金属業界の命題**: 「**一次資源依存 → 都市鉱山×次世代材料の二本柱への転換**」（みずほ産業調査2025-05）
2. **製缶業界の命題**: 「**サステナブル包装プラットフォーム**」 — 脱プラ追い風 + 国内縮小の二律背反
3. **時事フック**: 中国レアアース規制 **2025年4月強化** / EU PPWR **2026年8月適用** / みずほ産業調査「非鉄金属」**2025年5月発行**
4. **大企業 vs 中堅中小の経営戦略議論の本質差**: 「**自社をどう生き残らせるか**」vs「**業界ごとどう変革するか**」（業界変革プレイヤーとしての視座）
5. **「業界としてのありたき姿」の具体**: 個社では成し得ない業界協調論点（リサイクル基盤・トレーサビリティ・脱炭素ロードマップ・国際ルール形成）

## 1. 貴金属業界

### プレイヤーマップ

| 階層 | 主要企業 | ポジション |
|------|---------|----------|
| **上流（鉱山権益）** | **住友金属鉱山**（売上1.1兆円） | 菱刈金鉱山・カナダCôté・フィリピンCoral Bay。世界クラス鉱山権益保有の唯一の日本企業 |
| **中流（精錬）** | **三菱マテリアル**（1.6兆円）/ **DOWA HD**（4,400億円） | 銅製錬国内最大手 / 環境×製錬×電子材料の循環型 |
| **下流・加工** | **田中貴金属G**（非上場、2025-01持株会社化）/ **JX金属**（2024年上場、ENEOS HD傘下） | 宝飾〜半導体材料 / スパッタリングターゲット世界60%・圧延銅箔80% |

### 5大経営課題

1. **資源確保 × 地政学リスク**
   - 中国レアアース（ジスプロシウム等）輸出規制 **2025-04強化**
   - コンゴ・チリ・豪州依存
   - JOGMEC協調 + G7重要鉱物行動計画（Kananaskis 2025合意）

2. **製錬マージンの構造崩壊**
   - 銅精鉱TC/RC（製錬手数料）が歴史的低水準
   - 三菱マテリアル: 中計2026-2028 で「**脱・銅精鉱依存**」を明示。E-Scrap処理量2035年度倍増・銅精鉱処理量を60-70%に削減

3. **脱炭素・グリーン化**
   - 精錬は電力多消費産業
   - 住友金属鉱山のScope3排出量は全社GHGの **67.8%**
   - JX金属「グリーンハイブリッド製錬」（酸化反応熱で化石燃料ゼロ化）が先進事例

4. **Urban Mining（都市鉱山・E-Scrap）**
   - 日本は貴金属蓄積量で世界有数の「都市鉱山大国」
   - EV普及で2025年以降に大量のLiB廃棄開始
   - 欧州電池規則（**2030年以降にリサイクル材最低含有率を義務化**）が追い風

5. **次世代材料（EV・半導体・水素）**
   - EV/全固体電池: 三菱マテリアルがxEV用全固体電池向け材料の新製造技術
   - 半導体: JX金属のスパッタリングターゲット（AI向けデータセンター需要急増）
   - 水素触媒: 燃料電池触媒に白金（Pt）・パラジウム（Pd）必須
   - 磁性材料: EV駆動モーター向けNd磁石

### 各社の戦略ポジショニング

| 企業 | キーワード | 中計の核心 |
|------|---------|----------|
| 住友金属鉱山 | 鉱山権益・EV材料 | 中計2027「世界の非鉄リーダー」当期利益1,500億円/年 |
| JX金属 | AI・半導体・グリーン製錬 | ハイテク素材企業ポジション |
| 三菱マテリアル | サーキュラーエコノミー | 中計2026-2028「E-Scrap倍増・資源循環」 |
| DOWA | 廃棄物→資源・都市鉱山 | 中計2027「循環のクオリティ」営業利益470億円・ROE10%以上 |
| 田中貴金属 | 宝飾〜産業材料一気通貫 | 2025-01持株会社化 |

### 業界として目指すべき姿（みずほ産業調査2025-05より）

> 「**リサイクルを通じた鉱物資源の確保**」が業界命題

**業界の方向性**: 一次資源依存 → 都市鉱山×次世代材料の二本柱への転換

**業界協調が必要な論点**（個社では成し得ない）:
1. **業界横断 E-Scrapリサイクル基盤の構築**: 廃電子機器・廃電池の回収ネットワーク
2. **トレーサビリティ共通プラットフォーム**: 紛争鉱物フリー・カーボンフリー証明（欧州CBAM・電池規則対応）
3. **業界共通の脱炭素ロードマップ**: グリーン電力優先供給・電力コスト支援の政策提言
4. **政府への要請**: JOGMEC予算拡充・重要鉱物備蓄支援・二次原料製錬への設備投資補助

## 2. 製缶業界

### プレイヤーマップ

| 階層 | 主要企業 | ポジション |
|------|---------|----------|
| **国内ダントツ首位** | **東洋製罐グループHD**（連結9,600億円見通し FY2026） | 東洋製缶 + 東洋鋼鈑 + 東洋アルミ + 東洋ガラスの総合包装 |
| **専業大手** | 大和製罐（非上場・伊藤忠系）/ ホッカンHD（936億円）/ アルテミラ製缶（旧ユニバーサル製缶・麒麟系） | アルミ・スチール缶・エアゾール缶 |
| **国際比較** | Ball Corporation（118億ドル）/ Crown Holdings（120億ドル） | 日本勢は国内特化で国際競争力限定 |

### 市場規模

国内金属缶業界 **約1.6兆円** / 世界アルミ缶市場 **約400億ドル**（2026年）/ 年率2-5%成長

### 5大経営課題

1. **脱プラスチック規制への対応**
   - **EU PPWR**（包装・包装廃棄物規則）2024-11採択・**2026-08適用開始**
   - 食品缶 BPAフリー塗料切替も義務化（EU 2025-01施行）
   - 日本: 容器包装リサイクル法改正 2026-04 再生材利用義務拡大
   - 「**脱プラの受け皿として金属缶が追い風**」の側面

2. **軽量化・薄肉化によるCO2削減**
   - 東洋製缶 × 新日鉄住金「**世界最軽量飲料スチール缶**」（従来比40%軽量化）
   - 東洋製罐G GHG目標: **Scope1,2 を 2030年度 2019年度比50%削減 / Scope3 30%削減**

3. **リサイクル率向上と資源循環**
   - **アルミ缶リサイクル率 99.8%（2024年・世界最高水準）**
   - スチール缶 93%以上
   - 次の課題: **「ボトル to ボトル」（缶から缶への完全閉ループ）**

4. **国内人口減少 → 海外展開**
   - 東洋製缶 + ホッカン: 2016年経営統合基本合意（コスト競争力・海外強化）
   - タイ・ミャンマー整理（東洋製缶は2024年にタイ子会社議決権25.1%譲渡）
   - ホッカンの海外投資 **117億円/年**（2025年計画）

5. **機能性・高付加価値化**
   - BPAフリー内面塗装の全面切替
   - 窒素充填技術（コーヒー飲料アルミ缶対応）
   - 医薬品・健康食品向け（長期保存・遮光性）
   - 小ロット・デザイン対応（クラフトビール・D2Cブランド向け）

### 業界として目指すべき姿

**サステナブル包装プラットフォーム** — 缶を「素材」から「サーキュラーソリューション」へ再定義。リサイクル率世界最高水準を維持しつつ、脱プラ需要の受け皿として缶の優位性を社会に訴求。

**業界協調が必要な論点**:
1. **缶リサイクルのクローズドループ確立**（業界・飲料メーカー・自治体の三者協調）
2. **業界共通カーボンニュートラル宣言**（グリーンアルミ・グリーンスチールの調達基準）
3. **国際標準への参画**（EUサステナブル包装基準にルール・メイカーとして関与）
4. **政府への要請**（容器包装リサイクル法のEPR明確化・LCA基準策定）

## 3. 中堅中小企業 vs 大企業の経営戦略議論の質的差

### 意思決定プロセス

| 軸 | 中堅中小企業 | 大企業 |
|----|------------|--------|
| 意思決定者 | オーナー社長・創業家 | 取締役会・経営企画・社外取締役・株主 |
| 承認プロセス | ワンマン決定が多い | コーポレートガバナンス・IR介在 |
| 情報非対称性 | 高（社長が全情報保有） | 低（開示義務・IR説明責任） |
| 戦略の公開度 | 非公開が基本 | 中計・統合報告書・サステナビリティ報告書で公開 |

### 時間軸

| 軸 | 中堅中小 | 大企業 |
|----|---------|--------|
| 主な計画軸 | 3-5年の短中期 | **中計3-5年 + 長期ビジョン10-30年 + パーパス（永続）** |
| 事業承継視点 | M&Aエグジット・後継者問題が最重要 | 事業ポートフォリオ最適化・**業界再編プレイヤー** |
| 投資判断 | 「回収できるか」「借入できるか」 | 「**ROIC・ROCE・資本コストを上回るか**」 |

### 「経営戦略のディスカッション」の中身

**中堅中小RMの典型論点**:
- 次の3年でどこで成長するか（販路拡大・設備投資）
- 後継者・M&Aで売るか継続するか
- 金利・借入条件・BS改善
- 補助金・政策支援の活用

**大企業RMの典型論点**:
- CFOクラスと **資本政策・格付け戦略** を議論
- **サステナビリティ・グループ再編** といった経営上流
- **業界の構造変化（脱炭素・地政学・技術革新）に対してどうポジションを取るか**
- **10年後の業界構造の中で自社は何を取るか**
- **業界全体のリーダーとして果たすべき役割**

### 本質的な違い

| 中堅中小 | 大企業 |
|---------|--------|
| **自社をどう生き残らせるか** | **業界ごとどう変革するか** |
| 個社最適 | 業界最適 + 個社最適 |
| 融資直結の話が中心 | **融資に直結しない話（M&A・海外戦略・人的資本）** から将来事業機会が生まれる前提 |
| RM = 取引相手 | RM = **「業界変革の伴走者」**としての長期関係性 |

→ 大企業RMは「**業界が10年後にどうあるべきかをCFO・経営企画と議論できる**」ことが本質的やりがい

## 4. 「業界としてのありたき姿」具体イメージ

### 貴金属業界 10年後（2035年）

**業界構造の予測**:
- 一次精錬（銅精鉱処理）は縮小または海外移転
- 日本の精錬能力は **高品質二次原料（E-Scrap・廃電池）処理に特化**
- EV/全固体電池・AI半導体・水素燃料電池の三大テクノロジー向け材料が主戦場
- 都市鉱山からの回収が「第二の鉱山」として確立
- 中国依存の重希土類は国産・代替技術で2030年代に供給安定化

### 製缶業界 10年後（2035年）

**業界構造の予測**:
- 国内缶市場は人口減少で漸減
- スチール缶のさらなる縮小 / アルミ缶の相対的安定（RTD・機能性飲料）
- 金属缶は「**リサイクル100%・BPAフリー・カーボンニュートラル**」3点セットでプラスチックに対する環境優位性確立
- グローバル成長エンジン = 東南アジア・インド・北米
- **Can-as-a-Service**（小ロット・デザイン対応・充填まで一気通貫）の新ビジネスモデル台頭
