### ★ Vol.78 Ch.21「非鉄金属 ～リサイクルを通じた鉱物資源の確保に向けて」（2025-05-30）

**テーマ**: 経済安全保障上の重要鉱物（Li・Ni・Co・レアアース）の安定供給をリサイクルで確保

| 論点 | 要旨 |
|------|------|
| 背景 | CN向け蓄電池・モーター需要拡大 → 重要鉱物はほぼ全量輸入。経済安全保障推進法で35鉱種を特定重要物資指定 |
| LiBリサイクル現状 | 技術確立済みだが現状は工場端材のみ。2040年には廃LiBから需要の30%を回収可能と試算 |
| 3大課題 | ①リサイクルコスト高（運送・中間処理エネルギー・精製）②廃LiB量確保（中古車輸出で国外流出）③欧州電池規則対応（トレーサビリティ） |
| 主要プレイヤー | 住友金属鉱山（精製主力、2026年6月プラント完成）、JX金属（敦賀新会社）、三菱マテリアル（パイロット）、DOWA（東南アジア展開）、エンビプロHD |
| みずほ弊行仮説 | ①Hub&Spoke構想（エリア毎に中間処理+精製拠点）②精製企業の収益化（川上延伸 or 加工賃モデル）③海外廃車回収拠点 |
| ネオジム磁石 | LiBと同様の打ち手。三徳・信越化学・エンビプロHD×HyProMag。事業化見通しは立たず |
| 欧州電池規則 | 2025年末までにLiB重量ベース65%リサイクル義務。2030年末70%。バッテリーパスポート・DD義務 |

**著者**: 産業調査部 素材チーム 杉本透（toru.sugimoto@mizuho-bk.co.jp）

**面接での活用**: #4「産調レポートの活用実態」で「Vol.78の非鉄金属リサイクル編を拝読した。住友金属鉱山の2026年6月プラント完成は貴部のお客様にも直接影響があると思うが、こうしたレポートをRM側ではどのように活用されているか」

### ★ Vol.79「日本産業の中期見通し ー向こう5年（2026-2030年）の需給動向と求められる事業戦略ー」（2025-11-28）

**最重要**: 全91ページ・18産業カバー。年1回の旗艦レポート。

**非鉄金属セクション要点（P21-24）**:

| 項目 | 内容 |
|------|------|
| グローバル需要 | 2026年29,836千t（+2.0%）。EV・データセンター・インフラで2030年まで年率+2.0% |
| 国内需要 | 2026年864千t（+0.8%）。電子部品向けが牽引。建設向けは減少 |
| 買鉱条件 | TC/RCが需要・供給双方の要因で低水準継続。銅製錬企業にとって厳しい局面 |
| リスク | 中国・アジアの銅内製化 → 日本の電気銅輸出減。銅価格高騰で他素材代替リスク |
| チャンス | EV化・AI・データセンター向け高機能伸銅品需要拡大。電線需要増 |
| 戦略方向性 | 鉱石に依存しないリサイクル原料活用への転換。民間と政府の協調 |

**鉄鋼セクション要点（P17-20）**: 保護主義強化 → 輸出減 → 現地鉄鋼一貫生産体制構築。内需縮小＋構造改革。

**産業総合（P2-4）**: 5つの外部環境変化（①国際情勢緊張②脱炭素揺らぎ③供給制約④人口動態⑤デジタル化）→ 3戦略（供給体制適正化 / ビジネス領域拡張 / デジタル技術活用）。非鉄金属は「供給体制適正化」の対象産業。
