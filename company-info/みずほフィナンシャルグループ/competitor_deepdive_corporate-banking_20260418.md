## 軸7: りそな「銀信一体」vs みずほ「別法人連携」の構造比較（★最重要）

### りそなの「信託併営」とは具体的に何か

りそな銀行は**旧大和銀行（旧野村銀行）の流れを汲む、戦前から信託併営を続ける唯一の商業銀行**。以下の信託機能が銀行本体に内蔵されている:

| 信託機能 | りそな銀行（内蔵） | みずほ（別法人TB経由） |
|---|---|---|
| 自社株承継信託（議決権留保型） | **銀行RMが直接提案・実行** | BK RM → TB担当者へ引き継ぎ |
| 自社株承継信託（遺言代用型） | **銀行RMが直接提案・実行** | BK RM → TB担当者へ引き継ぎ |
| 不動産仲介 | **銀行RMが物件紹介→仲介→融資まで一気通貫** | BK RM → TB不動産部門へ引き継ぎ |
| 遺言信託 | **銀行本体で受託** | TB経由 |
| 事業承継コンサル | 承継ソリューション営業部（約200名）が銀行内で完結 | BKコンサル部＋TBコンサル部の連携 |

### みずほ「BK→TB連携」で生じる構造的コスト

- 別法人間の**情報共有には顧客同意が必要**（銀行法上の規制）
- 担当者が変わるため**顧客との関係構築が分断**されるリスク
- 社内稟議・案件引き継ぎに**時間がかかる**（推測: 数日〜数週間）
- TB側の専門性は高いが、**BK RMが信託の全体像を把握しにくい**

### りそなにできてみずほにできないこと

| 項目 | 説明 |
|---|---|
| **RM1人でのワンストップ提案** | りそなRMは融資→信託→不動産仲介を自分で完結できる。みずほRMは信託部分をTBに引き継ぐ必要がある |
| **スピード** | 別法人間の手続きが不要なため、意思決定から実行まで早い |
| **顧客体験の一貫性** | 担当者が変わらないため、オーナーとの信頼関係が維持される |
| **低コスト層への信託提供** | PB最低1億円〜で中堅オーナーにも信託サービスを展開。みずほPBは10億円〜（推測） |

### みずほにできてりそなにできないこと

| 項目 | 説明 |
|---|---|
| **証券機能（M&A・IPO・DCM）** | みずほ証券GIBの投資銀行機能はりそなに存在しない。事業承継の先のM&A・IPO・社債発行まで一気通貫 |
| **海外進出支援** | みずほは約40都市に直営拠点。りそなは駐在員事務所6拠点（上海・香港・バンコク・ホーチミン・ハノイ・LA） |
| **産業調査機能** | みずほ産業調査部の業界知見はりそなにない。りそな総研は地域経済・中小診断が中心 |
| **大型案件への対応** | メガバンクの信用力・バランスシート規模で大型融資・シンジケートローンに対応。りそなは中堅中小特化 |
| **スタートアップ支援の幅** | UPSIDER/BLUE DREAM Fund/みずほ証券IPOの一気通貫。りそなはベンチャーデット開始直後 |
| **グループ会社の専門性** | みずほ信託銀行は**信託特化度が大手信託銀行中最高**（手数料収入比率が資金利益の2倍近く）。りそなの信託は銀行業務の付随機能 |

### 「りそなでいいのでは？」への具体的反論ロジック

> 「りそなの銀信一体はワンストップのスピードで優れています。ただ、中堅中小企業の経営課題は事業承継だけではありません。**海外進出・M&A・IPO・社債発行・産業構造の転換**といった成長局面では、みずほの証券・信託・産業調査部のプロフェッショナルを組み合わせた提案が必要になります。りそなは『承継のスピード』、みずほは**『承継の先の成長戦略まで含めた幅』**。私は後者を提供できるRMになりたいと考えています」

> 「もう一つ付け加えると、みずほ信託銀行は**手数料収入比率が大手信託銀行中最高で、信託の専門性が最も高い**。りそなの信託は銀行業務の延長で提供される『便利な付加機能』ですが、みずほTBは**信託のプロが専任で対応する深い提案**ができます」

| 中計目標 | 純利益1,200億円超（2028年3月期） | FFGベンチャー累計500億円規模 | みずほFG純利益1兆円超 |
| デジタル | 実店舗＋ネット銀行対抗の「利便性・安心感」 | **みんなの銀行（BaaS）**＋Wallet+ | UPSIDER＋One MIZUHO |
| 弱み | グループ機能の幅が限定的。海外・M&A・IPOは外部依存 | 同左 | 地域密着度・オーナーとの長期関係 |

### 地銀FGにできてメガにできないこと

- **同一地域での長期RM配置**: 5年以上同じ顧客を担当し、オーナーの人生に伴走するスタイル
- **地域経済との一体感**: 地元企業同士のマッチング、地域イベントを通じた関係構築
- **小規模融資への丁寧な対応**: 数百万円単位の融資にも専任担当がつく

### メガ（みずほ）にできて地銀にできないこと

- **グループ機能の幅**: 証券（M&A・IPO・DCM）、信託（承継・不動産）、産業調査、海外40都市のネットワーク
- **大型案件への橋渡し**: 中堅中小企業が成長して大企業になる過程で、シンジケートローン・社債・海外進出を途切れなく支援
- **UPSIDER/BLUE DREAM Fundによるスタートアップ支援**: 地銀FGのVCは地域限定的だが、みずほは全国・全産業をカバー
- **産業横断の知見**: みずほ産業調査部の全産業カバレッジ。地銀シンクタンクは地域経済レポートが中心

### 「地銀でいいのでは？」への具体的反論ロジック

> 「地銀FGの地域密着は本当に素晴らしいと思います。ただ、中堅中小企業が成長して海外進出やM&A、IPOを目指す段階になると、**地銀のグループ機能では対応しきれず、結局メガバンクに案件が流れる**のが現実です。私は最初から**成長の全ステージに伴走できるメガのRM**として、地銀では提供できない『承継の先の成長戦略』を届けたい」

> 「特にみずほの場合、**産業調査部がRMと一緒に取引先に入って戦略提案を行う仕組み**があります。地銀シンクタンクは地域経済レポートが中心ですが、みずほは**全産業×全地域の知見をRM提案に直結**させられる。これは地銀にはないメガ固有の武器です」
