# みずほ銀行 中堅・中小企業RM 業務理解 詳細レポート

> 目的: Career Lounge「中堅・中小企業RM」面談前に、職種・業務・組織・最新動向を深く理解し、面談で「業界理解の深さ」を示し、質の高い逆質問を出すための土台を作る。

---

## 0. エグゼクティブサマリー

- **RM（Relationship Manager）** = 法人営業の主担当。お客様企業の経営課題を起点に、融資・決済・M&A・事業承継・海外展開・IT/DX等の **金融＋非金融ソリューション** を提供する。
- **「中堅・中小企業RM」** は、年商数十億〜数百億円程度のオーナー企業・地場優良企業を担当するRM。**全国の支店配属**が基本で、地域密着と経営者との距離の近さが特徴。
- **大企業RMとの違い**: 大企業RMは本部組織・大型案件・グローバル中心。中堅中小RMは**支店・全国網・経営者との直接対話**。
- **2021年に大改革**: みずほ銀行は業種別のいわゆる「ナンバー部」（営業第一部、第二部…）を廃止し、**業種横断型の5グループ体制**に再編。背景は「業態を超えたM&A」の増加。
- **戦略の軸**: 事業承継（**2025年までに中小経営者の約64%が70歳超**）、M&A、海外進出支援、DX支援、One MIZUHO（銀証信託一体）。

## 1. RM（リレーションシップマネージャー）とは

### 1-1. 役割
- 法人顧客の **窓口かつ司令塔**
- 顧客の経営課題・財務戦略を理解し、最適な金融ソリューションを企画・提案
- 行内のプロダクト部署（M&A・不動産・国際・市場・信託・証券）を巻き込み案件を組成
- **顧客との長期リレーション構築**が最大のミッション

### 1-2. 「営業」と「RM」の違い
- 一般的な営業: 商品を売る
- RM: **顧客企業の戦略パートナー**として、課題発掘 → ソリューション組成 → 実行 → アフターフォローまで一貫対応

## 2. 中堅・中小企業RMの業務内容

### 2-1. 対象顧客
- 年商: 概ね **数十億〜数百億円規模**（大企業未満・小規模事業者超）
- 地域: 全国の支店エリアの地場優良企業・オーナー企業
- 業種: 製造業・卸売業・小売業・サービス業・不動産業など多岐
- 取引深度: **既存深耕＋新規開拓**

### 2-2. 提供する金融ソリューション

| カテゴリ | 内容 |
|---------|------|
| **融資・資金繰り** | 運転資金・設備資金・コミットメントライン・シンジケートローン |
| **決済・為替** | 国内送金・外為・キャッシュマネジメント |
| **資本性ファイナンス** | メザニン・劣後ローン・私募債 |
| **事業承継** | 後継者対策・自社株対策・M&A仲介 |
| **M&A** | 譲渡・譲受双方のアドバイザリー（みずほ証券・グループM&A会社と連携） |
| **海外進出支援** | 現地法人設立・クロスボーダー融資・貿易金融 |
| **DX/IT** | みずほR&Tやグループ企業との連携 |
| **ビジネスマッチング** | 取引先紹介・販路拡大 |
| **資産運用** | 余資運用、社長個人の資産形成（PB連携） |
| **信託機能** | 不動産・遺言・年金（みずほ信託と連携） |

### 2-3. 1日の業務イメージ（体験記ベース）

| 時間帯 | 業務 |
|-------|------|
| 朝 | 朝礼 → メール・市況確認 → 案件資料準備 |
| 午前 | 顧客訪問（融資打診・進捗確認・経営者面談） |
| 昼 | 顧客と会食 or 内勤 |
| 午後 | 顧客訪問2〜3件 → 帰店後に稟議・案件資料・本部相談 |
| 夕方 | 上司報告・翌日アポ確認・残務 |

> 担当先数: 体験記によると **若手RMで20〜40社程度**を担当するケースが多い。

### 2-4. 新規開拓の難しさ
> 「中堅・中小企業RMの新規開拓は、ゼロからお取引をスタートさせる**非常に厳しい世界**。お客様も気づかない経営課題や潜在ニーズを発掘し、みずほにしかできない提案をすることが重要」（公式採用ページ）

## 3. 大企業RM vs 中堅中小RMの違い

| 項目 | 大企業RM | 中堅・中小企業RM |
|------|---------|---------------|
| 対象 | 上場大企業・グローバル企業 | 地場中堅・オーナー企業 |
| 配属 | 本部営業部 | 全国の支店 |
| 案件規模 | 大（数百億〜数千億） | 中（数億〜数十億） |
| 対面相手 | 経理部長・財務部長・CFO | **オーナー社長・専務クラス** |
| 案件の性格 | 資本市場系・グローバル系 | 事業承継・M&A・地域密着 |
| キャリア成長 | 専門性重視 | **経営者と深く話せる総合力** |
| 雰囲気 | フォーマル | 距離が近くアットホーム |

## 4. みずほ銀行の組織体制（最新）

### 4-1. 大改革（2021年4月〜）
- 従来: 業種別の **「ナンバー部」**（自動車=第9部、電機=第10・11部 等）
- 新体制: 業種横断の **5グループ** に再編
- 背景: 「業態を超えたM&A・提携の増加で旧来の業種別では非効率」（日経）

### 4-2. リテール・事業法人カンパニー（個人＋中堅中小法人）
- 個人と中堅中小法人を一体運営する **カンパニー制**
- 全国の支店ネットワークが営業基盤
- 中堅中小企業RMはこのカンパニー所属

### 4-3. 大企業法人カンパニー
- 大企業向け。本部営業組織。
- 大企業RM・産業調査・プロダクツ・ソリューション部隊で構成

### 4-4. グローバルプロダクツユニット（GPU）
- M&A・ストラクチャード・国際・市場プロダクトを横串で提供
- RMが顧客課題を持ち込むと連携

### 4-5. みずほ銀行 × みずほ信託 × みずほ証券（One MIZUHO）

```
お客様（中堅中小企業）
        │
   [みずほ銀行 RM]  ← 窓口
        │
        ├──→ みずほ証券（M&A・社債・株式公開）
        ├──→ みずほ信託（不動産・遺言・年金）
        ├──→ みずほR&T（IT/DX・コンサル）
        └──→ プロダクツ部門（国際・市場・SF）
```

## 5. 若手RMのキャリアパス

### 5-1. 入社後の流れ（公式ベース）
1. **入社後1ヶ月**: 集合研修で業務基礎知識習得
2. **配属**: 全国の支店（営業店）
3. **OJT**: 先輩RMのサポート、与信関連業務、契約書作成、資料準備
4. **半年後**: **一人で顧客担当開始**
5. **3〜5年目**: 担当先を持ち、法人RMとして案件主担当
6. **5〜10年目**: 大企業RMへの異動 or 専門領域（M&A・国際・市場・本部企画）への展開
7. **10年目以降**: 課長・支店長・本部部長

### 5-2. キャリアの広がり（実例）
- 中堅中小RM → 大企業RM
- 中堅中小RM → M&Aアドバイザリー部
- 中堅中小RM → みずほ証券リテール・事業法人部門への出向
- 中堅中小RM → みずほ信託で事業承継・不動産案件
- 中堅中小RM → 海外駐在（アジア・北米拠点）
- 中堅中小RM → 産業調査部・本部企画

### 5-3. 体験記が語る若手の声
- 「30社程度のクライアントを担当し、リレーション構築力と財務分析の基礎を磨いた」
- 「20代でオーナー社長と直接話す機会が圧倒的に多い」
- 「**配属ガチャ**はあるが、入社後の研修・OJTは手厚い」

## 6. 中堅中小企業マーケットの社会的論点（面談で必須）

### 6-1. 事業承継問題
- **2025年までに中小企業経営者の約64%（約245万人）が70歳超**
- うち**約半数（127万人）が後継者不在**
- → みずほにとって **事業承継・M&Aは最大の事業機会**

### 6-2. 地域経済の構造変化
- 地方経済縮小・取引先減少 → 顧客の海外展開支援ニーズ
- 後継者不在 → M&A仲介・PE/サーチファンドとの連携

### 6-3. DX/IT支援
- 中小企業のIT人材不足 → みずほR&Tと連携したコンサル提案
- インボイス・電帳法等の制度対応支援

### 6-4. 競合との比較
- **競合**: 三菱UFJ銀行、三井住友銀行（メガバンク）／地銀／信金／日本政策金融公庫
- **みずほの強み**: 銀証信託一体、産業調査機能、One MIZUHO、楽天証券提携
- **みずほの課題**: 大企業偏重イメージの払拭、地方支店の収益性

## 7. 求められる人材像

### 7-1. スキル面
- 経営者と対等に話せる **総合力・教養**
- 財務分析・与信判断
- 提案構築力（顧客課題 → ソリューション設計）
- 巻き込み力（行内プロダクツ・グループ会社との連携）

### 7-2. マインド面
- **顧客第一**（売り込みではなく課題解決）
- **粘り強さ**（新規開拓は厳しい世界）
- **変化対応力**（事業承継・DX等の新領域）
- **学習意欲**（業界知識・税務・法務）
