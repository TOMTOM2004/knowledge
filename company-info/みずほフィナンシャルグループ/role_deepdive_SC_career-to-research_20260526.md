# みずほ証券 リサーチ/アナリスト職 と 法人WM営業からの異動経路

## 1. リサーチ/アナリスト系職種の正式名称（リサーチ&コンサルティングユニット傘下）

| 部署（正式） | 役職名 | 業務 |
|---|---|---|
| **エクイティ調査部** | セクターアナリスト／ストラテジスト／エコノミスト／クオンツアナリスト | 個別企業・業界・経済の調査分析。機関投資家向け株式リサーチ。日経ヴェリタス2025人気アナリスト調査3位 |
| **金融市場調査部** | 債券アナリスト／エコノミスト／為替アナリスト／クレジットアナリスト | 機関投資家向け債券投資戦略・マクロ・クレジット・ESG |

- 昇進ラダー: **アナリスト → アソシエイト → VP → ディレクター → ED**
- ※**みずほR&T（リサーチ&テクノロジーズ）は別法人**のシンクタンク（IT/DX/政策提言）。エクイティリサーチはやらない＝混同注意。銀行の**産業調査部(MRA)**ともまた別。

## 2. 【最重要】法人WM営業 → リサーチ への異動可否

**制度上は可能。現実は中〜高難度。「稀だが閉じてはいない」が正確。**

- 採用FAQ Q15: 初期配属は国内営業拠点 or 本社ホールセール → 「その後 本人の志向・能力・適性に応じてリサーチ&コンサルティングユニットなど様々な業務を経験する機会が開かれている」と明記
- **かなで（2024/7・5社横断人事）**: 社内公募が「部署主導」→「本人主導オープンエントリー」へ。年1,700人応募、リサーチは人気上位、異動実績あり、リスキリング支援（1人50万円）
- 口コミ: ジョブ公募は3年目から申込可／5年目以降リサーチに携わる例。ただし「支店→本社は相応の成果が必要」
- 現実的ルート: **営業実績＋CMA（証券アナリスト資格）早期取得 → 3〜5年で社内公募**

## 3. 新卒配属の一般則

- **キャリア特定型（スタート特定型）RCコース** = 入社時からリサーチ直配属（リサーチ志望の王道）
- **法人ウェルスマネジメントコース（グループオープン型）** = 初期はリテール・事業法人部門（営業）→ 社内公募で異動

## 4. 最終面接用：証券版 3段階キャリアパス（銀行版対比）

| 段階 | 銀行版（確立済み） | 証券版（確定） |
|---|---|---|
| Step1（〜5年） | 法人RM（融資・リレーション・ヒアリング） | **法人ウェルスマネジメント営業**（事業法人・オーナー企業のヒアリング） |
| Step2（5-10年） | 産業調査部 リサーチャー | **エクイティ調査部 セクターアナリスト**（かなで社内公募・CMA取得後） |
| Step3（10年〜） | GIB M&A・資本政策 | **GIB の ECM／M&Aアドバイザリー** |

## 5. 二択を貫く一本の論理（面接での語り）

- 銀行も証券も「リサーチ直行コースがあるのにあえて営業から入る」。理由は同じ＝**定性ヒアリング力は現場でしか磨けず、その純度がリサーチの質を決める**から（qa高8「なぜRMが先か」を証券にも適用）。
- 違いは到達先の制度的明確さのみ：**銀行＝産業調査部で明確／証券＝かなで公募で自ら切り拓く**（「だからこそ挑戦しがいがある」と前向きに）。
- 語り方の注意: 証券版は「制度上可能、かなでで現実性あり」と**正直に**。CMA早期取得など主体的準備で補強。「銀行でも証券でも同じ」と曖昧に流さず**"同じ軸・別ルート"**として説明する。

## 6. 銀行 vs 証券 決定版（差別化・優先順位）※最終面接の核

> ⚠ **最終面接の「銀行か証券か」一本化に直結**。次回オファー面接は1社に絞られるため、ここを言い切れる状態にしておく。

| 軸 | 銀行（本命） | 証券（次点） |
|---|---|---|
| キャリア弧 | 法人RM→産業調査部→M&A | 法人WM営業→エクイティ調査部→GIB M&A |
| **定性の取得源** | 当事者(経営者)から**継続的・深く・信頼関係下**＝最高品質 | 営業段階は当事者に入れるが、**リサーチ段階で市場/投資家寄りにズレる**（フェアディスクロージャー規制） |
| 定量の核 | 産業マクロ・業界構造 | 個別企業バリュエーション（純度はむしろ高い） |
| **ミクロ/マクロ** | **産業・業界・経済レベル＝マクロ寄り（本人の学業＝マクロ経済学・卒論・計量と一致）** | 個別企業＝ミクロ |
| 制度ルート | 産業調査部↔RM往復が明確＋かなで | かなで公募で自ら切り拓く |
| 顧客 | 法人・経営判断の最上流 | 機関投資家・個別企業価値・資本市場 |

### 優先順位＝銀行が本命（理由3＋1）
1. 自分の軸（一次情報の純度→定量×定性）に最も直結＝当事者の最深部の定性を取れるのは銀行RM
2. 制度ルートが明確（産業調査部↔RM）。証券は公募で切り拓く
3. 意思決定の最上流（法人の経営判断）に継続的に関われる
4. **マクロ志向との一致**：興味が個別企業(ミクロ)より産業・経済(マクロ)にある。卒論(輸入価格ショック→産業・家計波及)・ゼミ(計量経済)・学業(マクロ経済学中心)に裏打ち＝主観でなく一貫した軸。産業調査部(産業・業界レベル)がドンピシャ、エクイティ(個別企業ミクロ)とは志向が異なる

### マクロ/ミクロを使うときの注意
- 「マクロが好き」で止めず**卒論・ゼミ・学業を根拠に**（主観に見せない）
- マクロ志向は銀行強化に使う一方、証券を全否定しない（証券に振られたら「個別企業(ミクロ)を起点に産業(マクロ)へ繋ぐ／金融市場調査部のエコノミスト・ストラテジストはマクロ」と橋を架ける）

### 言い切り方
- **「銀行か証券か」→**「第一志望は銀行。"定性の一次情報を当事者から継続的に取れる"のは銀行RMで、分析のインプット品質を最大化する。加えて自分の興味は個別企業より産業・経済（マクロ）にあり、産業調査部と一致する」
- **「では証券は？」→**「同じ軸で語れる。法人WM営業で定性、エクイティ調査部で定量(バリュエーション)、GIB M&Aで統合。定量の純度はむしろ高く意思決定に直接効く。ただ"経営の最上流の一次情報"とマクロ志向で銀行を第一に置く」
- ❌ NG：「銀行でも証券でも同じ」（2次で言って濃淡が消えた）→ **"同じ軸・別ルート・優先は銀行"**で一貫
