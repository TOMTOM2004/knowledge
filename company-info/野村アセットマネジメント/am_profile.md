## C群: スチュワードシップ

### C-1 Stewardship integration ◎
- ESGスコアが株式・債券運用両方で投資判断・運用プロセスに統合
- ポートフォリオレベルのESGリスク管理: カーボンインテンシティ加重平均をベンチマーク対比低く、ESGスコア加重平均を高く
- **エスカレーション**: 3年経過で変化なしなら (1)社外取締役対話 → (2)議決権反映 → (3)**ダイベストメント/エクスクルージョン**
- エンゲージメント効果の定量測定（2024年分析）: TobinのQが3年で徐々に増大、統計的有意性確認
- **PRI 2024: 10項目中8項目★★★★★**（前年6項目から改善）
  - ポリシー97、上場株アクティブ・クオンツ100、アクティブ・ファンダメンタル100、パッシブ100、債券SSA 95、社債97、委託運用アクティブ94、委託運用債券100

### C-2 議決権行使方針・実績 ◎
- **2024年（1-12月）日本株式**
  - 会社提案 23,011件、**反対比率10.2%**
  - 取締役選解任 反対9.1% / 監査役11.2% / 役員報酬32.1%（ガバナンス対象拡大）
  - 退職慰労金97.6% / **買収防衛策100%**
  - 株主提案 369件、**賛成率17.1%**
- **外国株式**: 会社提案反対10.7%、株主提案賛成66.7%、合計反対11.9%
- **2024/11改定**: 
  - モニタリング・ボード要件厳格化（独立性要件全部満たす社外取締役過半）
  - ロールモデル基準にジェンダー多様性を追加（4課題目）
  - **ROE閾値引き上げ 5%→8%**（キャッシュリッチ企業）
  - **女性取締役最低水準 2025/11〜1名→10%に引き上げ**
  - 社外取締役人数: 支配株主いない=過半数

### C-3 対話テーマ・実績 ◎
- **日本株式**: **955件 2,491テーマ**、重点350社（保有額上位300+α）
- 投資先企業数 2,400社、中小型企業50社へ拡大
- **テーマ別**: ガバナンス31% / 事業戦略19% / 社会14% / 財務13% / 環境13% / 議決権10%
- **外国株式**: 508テーマ（延べ245件）+ Morningstar Sustainalytics 384件
- **マイルストン進捗（2024/12末）**: 完了411件、対応策実施69、対応策策定213、認識共有305、課題伝達302、**2024年100超が完了**
- 対話事例: 化学業界脱酸素連携、食品 日清製粉グループ（中計見直し）、台湾半導体、米国飲料サプライチェーン
- **重点テーマ9分類**（2024改編）: 成長戦略/資本効率/気候変動/自然資本/人権/人的資本/ウェルビーイング/**デジタル社会リスク（2024新設）**/モニタリング・ボード
