# dx-consulting-fit

## 前提
digital-common の観点を前提に、DXコンサル・変革推進職固有の評価基準を追加する。

## DXコンサル職で評価される特性

### 職種固有の評価軸
1. **変革推進・ステークホルダー巻き込み力**
   - 既存業務を変えることへの抵抗を乗り越えられるか
   - 現場・経営・ベンダーの三者を動かす調整力
2. **課題設定力（上流思考）**
   - 「何を作るか」より「何を解決するか」から考える
   - 要件定義・As-Is/To-Be整理の能力
3. **プロジェクトマネジメント**
   - スケジュール・コスト・品質・リスクの同時管理
   - 遅延・炎上時の対処経験
4. **コミュニケーション設計**
   - 経営層向け・現場向けで伝え方を変えられる
   - 「非エンジニアにも分かる言語化」

### ES・面接チェックポイント

**ガクチカ・自己PRで確認すること**
- [ ] 組織・チームを動かした経験があるか
- [ ] 複数の利害関係者が絡む問題を解決した経験があるか
- [ ] 「変化を起こした」実績が示せるか（小さくてもよい）

**よく聞かれる面接質問**
1. 「DXが失敗する原因は何だと思いますか？」
2. 「抵抗する現場社員をどう巻き込みますか？」
3. 「コンサルと事業会社DXの違いをどう認識していますか？」
4. 「テクノロジーと人の変化、どちらが難しいと思いますか？」
