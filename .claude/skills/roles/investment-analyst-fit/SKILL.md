# investment-analyst-fit

## 前提
securities-fit または asset-management-fit の観点を前提に、リサーチ職固有の評価基準を追加する。

## リサーチアナリスト職で評価される特性

### 職種固有の評価軸
1. **深掘り・一点突破の思考**
   - 幅広くではなく、1つのテーマを深く掘り下げる意志
   - 「専門家」になることへの適性・意欲
2. **数値モデル・定量分析への親和性**
   - 財務諸表の読解・バリュエーション計算
   - Excelでの数値整理・シナリオ分析
3. **文章・論文的記述力**
   - リサーチレポートは論理的構成・読者への配慮が重要
   - 結論→根拠→証拠の構成で書けるか
4. **業界・企業への長期的な関心**
   - 担当セクターを3〜5年単位で追い続ける継続力

### ES・面接チェックポイント

**志望動機で確認すること**
- [ ] 「深く1つのことを極める」志向が示されているか
- [ ] 分析・調査が「好き」だけでなく「継続できる」と示せているか
- [ ] レポート執筆・文章作成への苦手意識がないか

**よく聞かれる面接質問**
1. 「最近読んで印象に残った決算資料はありますか？」
2. 「あなたが担当したいセクターはどこですか？理由は？」
3. 「企業分析で最も重視する指標は何ですか？」
4. 「アナリストとファンドマネージャーの役割の違いをどう認識していますか？」
