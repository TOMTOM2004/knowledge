# fund-manager-fit

## 前提
asset-management-fit の観点を前提に、運用職固有の評価基準を追加する。

## ファンドマネージャー職で評価される特性

### 職種固有の評価軸
1. **投資判断の根拠構築力**
   - 「なぜこの株・債券を買うか」を論理的に説明できるか
   - 感覚ではなくプロセス・根拠で判断する習慣
2. **ポートフォリオ思考**
   - 個別銘柄ではなく「全体最適」で考える視点
   - リスク分散と収益最大化のトレードオフへの理解
3. **逆張り・独自判断への耐性**
   - 市場コンセンサスに反する判断を行う精神的強さ
   - 「みんなと違う意見を持ち、それを守れるか」
4. **長期コミットメント**
   - 投資は数ヶ月ではなく数年のスパンで評価される

### ES・面接チェックポイント

**ガクチカ・自己PRで確認すること**
- [ ] 「判断→実行→検証」のサイクルが経験として示せるか
- [ ] 数値・データをもとに意思決定した経験があるか
- [ ] 不確実な状況で決断した経験があるか
- [ ] 長期的なコミットメントの経験がある（3ヶ月以上続けたこと）

**よく聞かれる面接質問**
1. 「今注目している業界・企業はありますか？その理由は？」
2. 「あなたが今投資するとしたら、どこにどう投資しますか？」
3. 「市場が暴落したとき、どう判断しますか？」
4. 「アクティブ運用の存在意義は何だと思いますか？」
