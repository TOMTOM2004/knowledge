# motivation-answer-patterns

## 目的
ESに書いた志望動機を「面接での話し言葉」に変換する。
書き言葉（論理的・完結）と話し言葉（会話的・引き出しやすい）の違いを理解する。

## 話し言葉への変換ルール

### ESと面接の違い
| ES（書き言葉） | 面接（話し言葉） |
|-------------|--------------|
| 完結した論理構造 | 少し「余白」を残す（深掘りを誘う） |
| 1文が長くなりがち | 短い文で「間」を作る |
| 全部説明する | キーワードで話して、聞かれたら詳しく |
| 丁寧な文体 | 対話しながら修正できる柔軟さ |

### 面接での志望動機の構造（1〜2分版）

```
[冒頭のフック: 印象に残る1文]（15秒）
「私が○○業界を志望するきっかけは、〜という経験です。」

[Layer1: なぜこの業界か]（20秒）
「その経験から、〜に関わる仕事がしたいと考えるようになりました。」

[Layer2: なぜこの会社か]（30秒）
「その中で御社を志望する理由は、〜という点です。特に〜は他社にはない強みだと感じています。」

[Layer3: なぜこの職種か + ビジョン]（20秒）
「入社後は〜の職種で〜を実現したいと考えています。」
```

### 深掘りを誘う話し方
- 「特に〜という点に興味を持っています」→「具体的に教えてください」を誘える
- 「〜という経験から〜と感じました」→「その経験とはどんな経験ですか？」を誘える

## 面接官タイプ別の調整

| 面接官タイプ | 調整のポイント |
|-----------|-------------|
| 人事（hr-readable-style） | 一貫性・本気度を強調。企業名を具体的に使う |
| 若手社員（junior-humanities） | 分かりやすい言葉。親しみやすいトーン |
| 現場実務者（practitioner） | 業務理解の深さを示す。具体的な職種名で話す |
