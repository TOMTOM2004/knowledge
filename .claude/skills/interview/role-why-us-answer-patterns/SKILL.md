# role-why-us-answer-patterns

## 目的
面接で最も深掘りされる「なぜここか」の3階層の質問に対して、
論理的かつ感情的に説得力のある答えを話し言葉で組み立てる。

## 3階層の話し言葉での組み立て

### Layer1: なぜこの業界か
```
[原体験・きっかけ]（20秒）
「きっかけは〜という経験です。その時〜と感じました。」

[業界の機能との接続]（20秒）
「その経験から〜業界の持つ〜という機能・役割に関わりたいと考えました。」

[他業界との差別化]（20秒）
「製造業・コンサルタント等とも検討しましたが、〜という理由でこの業界が最も合っていると判断しました。」
```

### Layer2: なぜこの会社か
```
[業界共通点の認識]（10秒）
「この業界では各社が〜という強みを持っています。」

[この会社の固有性]（30秒）
「その中で御社を選ぶ理由は、〜という点です。
具体的には〜（競合には見られない固有の特徴）。」

[自分のビジョンとの接続]（20秒）
「この固有の強みが、私が実現したい〜に最も近い環境だと考えています。」
```

### Layer3: なぜこの職種か
```
[職種の業務理解]（20秒）
「〜職では〜を主な業務として〜を実現する役割だと理解しています。」

[強みとの対応]（20秒）
「私の〜という強みは、この職種での〜という場面で特に活きると考えています。」

[ビジョンとの整合]（20秒）
「長期的には〜になりたいと考えており、このキャリアパスが最も自然です。」
```

## 深掘り対処マトリクス

| 質問 | 対処パターン |
|-----|-----------|
| 「銀行でも証券でもなくAMを選んだ理由は？」 | 運用機能の独自性（受益者のために資産を育てる）を強調 |
| 「野村証券ではなくうちを選んだ理由は？」 | 競合比較（7-2マトリクス）から固有の1〜2点を示す |
| 「営業ではなくリサーチを選んだ理由は？」 | 業務内容への理解と自分の強みの対応を示す |
| 「本当に第一志望ですか？」 | 志望度の根拠を3点で示す |
