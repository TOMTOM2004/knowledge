# strengths-weaknesses-answer-patterns

## 強みの答え方

### 基本構造（30秒〜1分）
```
[強みの定義（1文）]
「私の強みは〜です。」（具体的なネーミングで）

[根拠となるエピソード（30秒）]
「例えば〜の場面で〜という行動をした結果、〜という成果がありました。」

[職種への接続（10秒）]
「この強みを○○職で〜という形で活かしたいと考えています。」
```

### 深掘りへの備え
- 「それって○○が得意なだけじゃないですか？」→ 「おっしゃる通りの側面もありますが、〜という点で○○とは異なります」
- 「他のメンバーと比べてどこが優れているのですか？」→ 具体的な比較エピソードを用意
- 「その強みが発揮できなかった場面は？」→ 弱みとセットで準備

## 弱みの答え方

### 基本構造（30秒〜1分）
```
[弱みの定義（1文）- 致命的でない弱みを選ぶ]
「弱みは〜です。」

[弱みが顕在化した場面（15秒）]
「〜という場面で〜という形で現れました。」

[対処策・改善（20秒）]
「そのため〜という取り組みをしています。現在は〜という状況です。」
```

### 弱みの「見せ方」の調整
- 深刻すぎる弱みは避ける（志望職種の本質能力に関わるもの）
- 「成長の余地がある」という文脈で語る
- 「直しています」より「こう対処しています」（進行形で）

## 面接官タイプ別の調整
| 面接官タイプ | 強みの見せ方 |
|-----------|-----------|
| 人事（HR） | 一貫性・成長性。ES記載内容と整合させる |
| 現場実務者 | 業務に直結する強みを前面に。専門性を示す |
| 役員（最終） | 大局観・ビジョン。「この人が30年後にどうなるか」の材料 |
