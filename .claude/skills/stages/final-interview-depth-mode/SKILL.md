# final-interview-depth-mode

## 最終面接（深掘り段階）の特徴
- 面接官: 部門長・役員・ベテラン社員（現場のプロ）
- 時間: 45〜90分
- 目的: 本気度の確認・論理の深さの確認・自社への適合性の最終判断
- 通過基準: 「この人を採用したい」という積極的な評価

## 最終面接で重視されること

### 1次と異なる評価軸
1. **深掘りへの耐性**: 「なぜ？」「具体的には？」に答え続けられるか
2. **自分の意見・判断**: 「どう思うか」に自分の考えを持って答えられるか
3. **企業・業界理解の深度**: 表面的でなく本質的な理解があるか
4. **長期ビジョンの整合性**: 会社のキャリアパスと自分のビジョンが合っているか
5. **覚悟**: 「この会社でいいのか」ではなく「この会社でないといけない」感

### 深掘りへの対処原則
- 準備した答えを丸暗記で話さない → 「考えている感」が大事
- 分からないことは「分からない。ただ〜と仮説を立てています」
- 反論されたとき: 即座に同意せず「確かにその点は〜ですが、〜と考えます」

## 最終面接の準備チェックリスト
- [ ] 「なぜこの会社でなければならないか」を3つ以上の根拠で語れる
- [ ] ガクチカを掘り下げた時（5段階の「なぜ」）に答えられる
- [ ] 「他に内定をもらったらどうするか」に答えられる
- [ ] 10年後のビジョンが企業のキャリアパスと整合している
- [ ] 業界・企業の課題について自分の見解を持っている

## よく聞かれる最終面接特有の質問
1. 「当社への最終的な志望度を教えてください」
2. 「他社と迷っている場合、当社を選ぶ決め手は何ですか？」
3. 「当社の課題は何だと思いますか？どう貢献できますか？」
4. 「入社後うまくいかなかった時、どうしますか？」
5. 「社会人として10年後どうなっていたいですか？」
