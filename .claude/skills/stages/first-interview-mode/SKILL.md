# first-interview-mode

## 1次面接の特徴
- 読者/面接官: 人事担当者 or 若手社員（1〜5年目）
- 時間: 30〜60分
- 目的: ESの内容確認・人物像の確認・志望度の確認
- 通過基準: 「問題なさそう」「もっと話を聞きたい」

## 1次面接で重視されること

### 確認項目（面接官の視点）
1. ESに書いてあることを自分の言葉で話せるか
2. 聞かれたことに答えられるか（設問適合）
3. 基本的な業界・企業理解があるか
4. 明らかなネガティブ要素がないか

### 話すべき内容の優先順位
1. **ガクチカ**: 最も聞かれる。3分で話せるように準備
2. **志望動機（なぜこの業界/会社）**: Layer1→Layer2の構造で
3. **自己PR**: 強みの根拠となるエピソード
4. **逆質問**: 2〜3問用意（「調べれば分かること」はNG）

## 1次面接の準備チェックリスト
- [ ] ESの内容を全部暗記でなく「話せる」状態にする
- [ ] ガクチカを1分・3分・5分の3バージョンで話せる
- [ ] 「なぜこの業界か」をLayer1（原体験）から語れる
- [ ] 「なぜこの会社か」を競合との比較で語れる
- [ ] 基本的な企業情報（事業・規模・競合）を把握している

## 1次面接で深掘りされやすいポイント
- ガクチカの「なぜその方法を選んだか」
- 「他の会社も受けていますか？」（正直に答えてよい）
- 「弱みは何ですか？」（短く・改善策とセットで）
