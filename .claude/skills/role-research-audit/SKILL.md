# role-research-audit

## 目的
志望職種に関する調査が面接・ES作成に使える品質かどうかを監査する。
「会社調査」と「職種調査」が適切に分離されているかを確認する。

## 使う場面
- 職種系設問（なぜこの職種か？入社後何をしたいか？）への回答前
- role-research-reviewerエージェントの内部手順として使う

## 職種調査チェックリスト

### 業務理解チェック
- [ ] 日次業務のイメージが具体的にある（○○を△△する）
- [ ] 職種内の役割分担・チーム構成がわかる
- [ ] 1年目・3年目・5年目で担う業務の変化がわかる

### スキル要件チェック
- [ ] 職種で求められる資格・専門知識が特定できている
- [ ] 定性スキル（コミュニケーション・分析等）の職種特有の意味がわかる
- [ ] 自分の強みとの対応関係が明確になっている

### キャリアパスチェック
- [ ] 職種内のキャリアラダーがわかる（担当→チーフ→マネージャー等）
- [ ] 職種外への異動可能性がわかる
- [ ] 会社固有のキャリア事例（OBOGの歩み等）がある

### 会社vs職種の分離チェック
- [ ] 「この会社だから」と「この職種だから」の理由が別々に説明できる
- [ ] 職種への関心が「会社が好き」とは独立して成立している

## 出力形式
- チェックリスト評価結果
- 不足項目リスト
- 追加調査クエリ案
