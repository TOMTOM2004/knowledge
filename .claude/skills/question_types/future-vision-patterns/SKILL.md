# future-vision-patterns

## 目的
「入社後何をしたいか」「10年後のビジョンは」「なりたい姿を教えて」系設問で、
企業のキャリアパスと整合した具体的・説得力あるビジョンを設計する。

## 基本構造

```
[短期（1〜3年目）: 何を身につけるか]
→ [中期（3〜7年目）: どんな専門家になるか]
→ [長期（10年後〜）: 何を実現したいか]
→ [なぜそのビジョンをこの会社で実現したいか]
```

## 設問別の答え方

### 「入社後何をしたいですか」（短期寄り）
→ 1〜3年目の具体的な学習目標と業務イメージを中心に構成

### 「10年後のビジョンを教えてください」
→ 長期像を中心に。ただし空中楼閣にならないよう短期との接続も示す

### 「当社でのキャリアをどう描いていますか」
→ 企業のキャリアパス（research_brief軸5）に沿ったロードマップを示す

## ビジョンの整合確認チェックリスト

- [ ] 企業のキャリアパス（research_brief軸5）と矛盾していないか
- [ ] 志望コースで実現可能なビジョンか
- [ ] 「なぜここでないとこのビジョンが実現できないか」の根拠があるか
- [ ] 他社でも言えるビジョンになっていないか
- [ ] 1年目から10年後まで連続性のあるストーリーか

## NGパターン
- 「管理職になりたい」（誰でも言える。何の専門家になりたいかが不明）
- 「グローバルに活躍したい」（具体性ゼロ。どの地域・どの機能か不明）
- 「社会に貢献できる人になりたい」（業界・職種と無関係なビジョン）
- 10年後の話だけして1年目・3年目の話がない（現実感がない）

## 業界別のビジョン設計のコツ

| 業界 | ビジョン設計のポイント |
|-----|-------------------|
| AM | 特定アセットクラスのプロ→ファンドマネージャー or チーフアナリスト |
| 銀行法人 | リレーションシップマネージャー→特定業種の専門家 |
| 証券リサーチ | 担当セクターのアナリスト→ファンドマネージャー等 |
| DX・IT | 業務・技術の専門家→変革リーダー |
