# motivation-question-patterns

## 目的
志望動機系設問（なぜこの業界/会社/職種か）の構造と答え方を整理する。
3つの「なぜ」を分離して、それぞれに根拠を持たせることが目標。

## 3階層の「なぜ」

### Layer 1: なぜこの業界か
- 他業界（製造・商社・コンサル等）ではなくこの業界を選ぶ理由
- 業界の機能・社会的役割への共鳴が必要
- 自分のルーツ・原体験との接続が強い

**構造テンプレート**:
```
[原体験・きっかけ] → [業界への関心の芽生え] → [自分のキャリアとの接続]
```

### Layer 2: なぜこの会社か
- 同業他社ではなくこの会社を選ぶ理由
- 競合との差別化ポイントを具体的に言える必要がある
- research_briefの軸7（競合比較）が根拠になる

**構造テンプレート**:
```
[業界への共感（Layer1）] → [この会社の固有特徴] → [自分のビジョンとの接点]
```

### Layer 3: なぜこの職種か
- 他の職種（営業・企画・ITなど）ではなくこの職種を選ぶ理由
- 職種の業務内容を理解した上での選択が示せるか
- 自分の強みとの対応関係が必要

**構造テンプレート**:
```
[職種の業務理解] → [自分の強みとの対応] → [この職種で実現したいこと]
```

## 設問別の答え方パターン

### 「志望動機を教えてください（400字）」
→ Layer1（1/4）→ Layer2（2/4）→ Layer3（1/4）の配分

### 「なぜ当社を志望しますか（200字）」
→ Layer2を中心に。Layer1・3は1文ずつ

### 「なぜこの職種を志望しますか」
→ Layer3中心。職種理解→強みの対応→ビジョン

### 「入社後何をしたいですか」
→ Layer3の具体版。1年目・3年目・10年目のイメージで構成

## よくある「なぜここか」の弱点と対処

| 弱点パターン | 対処 |
|-----------|------|
| 競合でも言える内容 | research_brief軸7-3で固有根拠を補強 |
| 業界への関心はあるが会社への関心が薄い | 競合比較を追加 |
| 感情的すぎて根拠がない | 数字・具体的な事実で補強 |
| 職種理解が浅い | role研究スキルで深掘り |
