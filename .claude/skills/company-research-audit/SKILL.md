# company-research-audit

## 目的
`company-info/<企業名>/research_brief.md` が面接・ES作成に使える品質かどうかを監査する。

## 使う場面
- ES作成・面接準備の前に調査品質を確認したいとき
- research-gap-reviewerエージェントの内部手順として使う

## 監査チェックリスト

### 必須項目（すべて揃っていること）
- [ ] 事業概要（主力事業・売上構成）
- [ ] 競合比較（主要3社との比較）
- [ ] 求める人材像（採用ページの文言）
- [ ] 直近の重要ニュース（最低3件）
- [ ] キャリアパス（入社後のキャリア形成）

### 推奨項目（ESに具体性を加える）
- [ ] 中期経営計画の概要と注力領域
- [ ] 会社固有の制度・文化・働き方
- [ ] OBOGからの一次情報
- [ ] IR資料からの業績トレンド

### 面接活用度チェック
- [ ] 「なぜこの会社か」を支える固有ポイントが3つ以上ある
- [ ] 「なぜ競合ではないか」を説明できる具体的な差別化点がある
- [ ] 「5年後どうなりたいか」に対応するキャリアパス情報がある
- [ ] 「最近の気になるニュース」に答えられる情報がある

## 品質スコアリング
- 必須5項目: 各2点（合計10点）
- 推奨4項目: 各1点（合計4点）
- 面接活用度4項目: 各2点（合計8点）
- **合計22点満点**

評価基準:
- 20点以上: 優秀（ES・面接に十分活用可能）
- 15〜19点: 普通（補足調査を推奨）
- 10〜14点: 不足（主要項目の追加調査が必要）
- 9点以下: 要再調査

## 出力形式
- チェックリスト評価結果
- 品質スコア（○/22点）
- 不足項目リスト
- 追加調査クエリ案（最大5件）
