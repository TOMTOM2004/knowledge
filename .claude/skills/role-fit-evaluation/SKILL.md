# role-fit-evaluation

## 目的
ESが志望職種の業務・スキル要件・キャリアパスと整合しているかを評価する。

## 使う場面
- 「この職種向けのESになっているか」確認したいとき
- 職種固有性が弱いと感じるとき

## 職種別評価マトリクス

### 資産運用系
| 確認項目 | 評価 |
|---------|-----|
| 金融市場・運用への関心の根拠 | |
| 数字・定量への志向性 | |
| 長期的思考・忍耐力の証拠 | |
| 情報収集・分析行動の実績 | |

### 銀行法人営業系
| 確認項目 | 評価 |
|---------|-----|
| 企業財務・経営課題への理解 | |
| 課題解決・提案行動の実績 | |
| 長期リレーション構築の志向 | |
| 粘り強さ・泥臭さの証拠 | |

### コンサル系
| 確認項目 | 評価 |
|---------|-----|
| 課題設定・仮説構築の実績 | |
| 構造化・論理的説明の能力 | |
| 主体的行動・変革意志の証拠 | |
| 多様なステークホルダー対応 | |

## 評価手順

### Step 1: 職種特定
志望職種を特定し、上記マトリクスの該当カテゴリを選択する。

### Step 2: ESから能力証拠を抽出
各チェック項目に対応する「証拠」をESから抽出する。

### Step 3: 対応関係マッピング
「職種要件 ↔ ESの証拠」の対応関係を表にまとめる。

### Step 4: 職種固有表現チェック
ESに職種特有の業務用語・職種名・職種固有の関心が言及されているか確認する。

## 出力形式
- 職種適合マトリクス評価
- 能力対応関係マップ
- 職種固有表現の有無
- 改善が必要な項目と補強すべき内容
