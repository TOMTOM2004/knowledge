# 選考段階別の評価基準・準備ガイド

面接の段階に応じた評価軸・チェックリスト・話し方の調整。

---

## ES書類選考段階

**読者**: 人事担当者、1枚3〜5分、同時期に数十〜数百枚と比較
**判断軸**: 説得力・具体性・一貫性・志望度の本気感

### 必須条件（1つでも欠けると即落選リスク）
- [ ] 各設問に正面から答えている（設問ズレなし）
- [ ] 禁止表現なし（御社・様々な・思います等）
- [ ] 敬称が統一（貴社/貴行/貴グループ）
- [ ] 字数制限を守っている（制限の90%以上）
- [ ] 未記入・[TODO]がない

### 推奨条件
- [ ] ガクチカに数字・定量表現がある
- [ ] 志望動機に企業固有の情報が含まれている
- [ ] 全設問を通じて人物像が一貫している
- [ ] 冒頭3行で「この人が何をした人か」が分かる

### 設問の優先順位
1. 志望動機（最重視） → 2. ガクチカ → 3. 自己PR → 4. その他

---

## 1次面接段階

**面接官**: 人事担当者 or 若手社員（1〜5年目）
**時間**: 30〜60分
**目的**: ESの内容確認・人物像の確認・志望度の確認
**通過基準**: 「問題なさそう」「もっと話を聞きたい」

### 確認項目
1. ESに書いてあることを自分の言葉で話せるか
2. 聞かれたことに答えられるか（設問適合）
3. 基本的な業界・企業理解があるか
4. 明らかなネガティブ要素がないか

### 話すべき内容の優先順位
1. ガクチカ（最頻出。3分で話せるように）
2. 志望動機（Layer1→Layer2の構造で）
3. 自己PR
4. 逆質問（2〜3問。「調べれば分かること」はNG）

### 準備チェックリスト
- [ ] ESの内容を「話せる」状態にする
- [ ] ガクチカを1分・3分・5分の3バージョン
- [ ] 「なぜこの業界か」をLayer1（原体験）から
- [ ] 「なぜこの会社か」を競合との比較で
- [ ] 基本的な企業情報（事業・規模・競合）を把握

### 深掘りされやすいポイント
- ガクチカの「なぜその方法を選んだか」
- 「他の会社も受けていますか？」（正直に答えてよい）
- 「弱みは何ですか？」（短く・改善策とセットで）

---

## 最終面接・2次面接以降（深掘り段階）

**面接官**: 部門長・役員・ベテラン社員（現場のプロ）
**時間**: 45〜90分
**目的**: 本気度の確認・論理の深さの確認・自社への適合性の最終判断
**通過基準**: 「この人を採用したい」という積極的な評価

### 1次と異なる評価軸
1. **深掘りへの耐性**: 「なぜ？」「具体的には？」に答え続けられるか
2. **自分の意見・判断**: 「どう思うか」に自分の考えを持って答えられるか
3. **企業・業界理解の深度**: 表面的でなく本質的な理解
4. **長期ビジョンの整合性**: 会社のキャリアパスと自分のビジョン
5. **覚悟**: 「この会社でないといけない」感

### 深掘りへの対処原則
- 丸暗記で話さない → 「考えている感」が大事
- 分からないことは「分からない。ただ〜と仮説を立てています」
- 反論されたとき: 「確かにその点は〜ですが、〜と考えます」

### 準備チェックリスト
- [ ] 「なぜこの会社でなければならないか」を3つ以上の根拠で
- [ ] ガクチカを5段階の「なぜ」で掘り下げ済み
- [ ] 「他に内定をもらったらどうするか」に答えられる
- [ ] 10年後のビジョンが企業のキャリアパスと整合
- [ ] 業界・企業の課題について自分の見解を持っている

### 最終面接特有の質問
1. 「当社への最終的な志望度を教えてください」
2. 「他社と迷っている場合、当社を選ぶ決め手は何ですか？」
3. 「当社の課題は何だと思いますか？どう貢献できますか？」
4. 「入社後うまくいかなかった時、どうしますか？」
5. 「社会人として10年後どうなっていたいですか？」
