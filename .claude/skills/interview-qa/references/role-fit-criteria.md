# 職種別の評価観点・面接質問

志望職種に応じたES・面接の評価軸・チェックポイント・頻出質問。

---

## ファンドマネージャー・運用職

### 職種固有の評価軸
1. **投資判断の根拠構築力**: 感覚ではなくプロセス・根拠で判断する習慣
2. **ポートフォリオ思考**: 個別銘柄ではなく「全体最適」で考える視点
3. **逆張り・独自判断への耐性**: 市場コンセンサスに反する判断を行う精神的強さ
4. **長期コミットメント**: 投資は数年のスパンで評価される

### ガクチカ・自己PRチェック
- [ ] 「判断→実行→検証」のサイクル経験
- [ ] 数値・データをもとに意思決定した経験
- [ ] 不確実な状況で決断した経験
- [ ] 長期的コミットメント（3ヶ月以上）

### 頻出面接質問
1. 「今注目している業界・企業はありますか？その理由は？」
2. 「あなたが今投資するとしたら、どこにどう投資しますか？」
3. 「市場が暴落したとき、どう判断しますか？」
4. 「アクティブ運用の存在意義は何だと思いますか？」

---

## 証券アナリスト・リサーチ職

### 職種固有の評価軸
1. **深掘り・一点突破の思考**: 1つのテーマを深く掘り下げる意志
2. **数値モデル・定量分析への親和性**: 財務諸表・バリュエーション・シナリオ分析
3. **文章・論文的記述力**: 結論→根拠→証拠の構成
4. **業界・企業への長期的な関心**: 担当セクターを3〜5年単位で追い続ける

### 志望動機チェック
- [ ] 「深く1つのことを極める」志向が示されているか
- [ ] 分析・調査が「好き」だけでなく「継続できる」か
- [ ] レポート執筆・文章作成への苦手意識がないか

### 頻出面接質問
1. 「最近読んで印象に残った決算資料はありますか？」
2. 「あなたが担当したいセクターはどこですか？理由は？」
3. 「企業分析で最も重視する指標は何ですか？」
4. 「アナリストとファンドマネージャーの役割の違いは？」

---

## データサイエンス・データ分析職

### 職種固有の評価軸
1. **ビジネス課題からのアプローチ**: 「何を解決するためにデータを使うか」
2. **実装・本番定着への意志**: PoC止まりではなく「動くシステムにする」コミットメント
3. **説明可能性への配慮**: 非データサイエンティストへの説明能力
4. **継続学習の姿勢**: 技術の変化スピードに追いつく自律学習習慣

### ガクチカ・自己PRチェック
- [ ] データを使って「何かを解決した」経験
- [ ] 手法の選定理由を説明できるか
- [ ] 非技術者に説明した経験

### 頻出面接質問
1. 「データ分析プロジェクトで最も苦労した点は？」
2. 「機械学習モデルを金融業務に適用する際の課題は？」
3. 「データの前処理で時間を使うことについてどう思いますか？」
4. 「AIの説明可能性（XAI）についてどう考えますか？」

---

## DXコンサル・デジタル戦略職

### 職種固有の評価軸
1. **変革推進・ステークホルダー巻き込み力**: 抵抗を乗り越える調整力
2. **課題設定力（上流思考）**: 「何を解決するか」から考える
3. **プロジェクトマネジメント**: スケジュール・コスト・品質・リスクの同時管理
4. **コミュニケーション設計**: 経営層向け・現場向けで伝え方を変える

### ガクチカ・自己PRチェック
- [ ] 組織・チームを動かした経験
- [ ] 複数の利害関係者が絡む問題を解決した経験
- [ ] 「変化を起こした」実績

### 頻出面接質問
1. 「DXが失敗する原因は何だと思いますか？」
2. 「抵抗する現場社員をどう巻き込みますか？」
3. 「コンサルと事業会社DXの違いをどう認識していますか？」
4. 「テクノロジーと人の変化、どちらが難しいと思いますか？」
