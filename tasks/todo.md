### 本セッション（2026-04-19）の核心的発見

**志望動機の構造転換**:
- 旧: 4固有性（会社の構造分析）が結論 → 「分析の精緻さ≠動機の強さ」問題
- 新: 本音の動機（複合判断の質を最大化したい）が結論 → 固有性は「証拠」として使う
- 動機の主語が「会社の特徴」→「自分のやりたいこと」に移った

**ユーザーの意思決定観（志望動機の核心）**:
- やりたいこと: さまざまな情報から複合的に一つの意思決定をする仕事
- 意思決定の3層: ①定量判断（論理的に矛盾がないか）②定性的見立て（自分の考えとマッチするか）③マーケット判断（誰にも読めない→チームで本音を出し合う）
- 「ISSに従うだけはありえない。そもそも除外して自分で考えている」
- 「なぜこの銘柄かを説明する側の人間」

**面談会一次情報の再解釈**:
- 「FMとアナリストで意見が割れることは普通にある」→ 忖度なく本音の見立てが出る環境
- ユーザーの感想: 「違う意見を言ってくれる人がいる環境、それだけで十分」
- FM迎合が避けられる環境 → 複合判断の質が上がる

**MYAMを選ぶ理由の3点（証拠として）**:
1. 少数精鋭49名×シニアPM15-19年 → 忖度が働きにくい → 本音の見立てが集まる
2. 営業運用分離 → アナリストが公平に銘柄を見ている → FMに上がる情報にバイアスなし
3. ISS非依存 → 判断を外注せずエンゲージメントに本気で向き合う姿勢

**少数精鋭の新しい解釈（ユーザー発）**:
- 大組織 → 一人の担当セクターが狭い → セクター間の関係・間接的影響が見えなくなる
- 少数精鋭 → 一人が広く見ている → 他社が見逃す関係やリスクが見える
- AI活用で広い範囲でも深さを保てば、大組織を超える分析が可能

**ESG-DCF統合を削除した理由**:
- ユーザー本人がESG投資をそこまで見ていない
- 投資家・公募投信の出資者はESGを見て投資していない人が多い
- マーケットのことを考えると全く関係ないというのが本音
